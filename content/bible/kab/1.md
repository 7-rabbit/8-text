      <li>Lḥakem yenna-yasen : Anwa ger sin-agi iwumi tebɣam ad serrḥeɣ ? RRran-as : I Barabas ! Serreḥ-ed i Barabas !</li>
      <li>Bilaṭus yenna-yasen : D acu ara xedmeɣ ihi i Ɛisa ițțusemman Lmasiḥ ? RRran-as-d akk : Ad ițțusemmeṛ ɣef wumidag !</li>
      <li>Lḥakem yenna : D acu i gexdem n diri ? RRnan țɛeggiḍen : Semmeṛ-it ɣef wumidag !</li>
      <li>Mi gwala lɣaci ugin a s-d ḥessen, yerna ccwal simmal yețțimɣuṛ, Bilaṭus yeddem-ed aman, issared ifassen-is zdat lɣaci, yenna : Nekk ur țekkiɣ ara di tazzla n idammen n uḥeqqi-agi, wagi d ccɣel-nwen.</li>
      <li>Lɣaci meṛṛa nnan-as : Tazzla n idammen-is a d-tuɣal ɣef yirawen-nneɣ d yirawen n warraw-nneɣ.</li>
      <li>Imiren Bilaṭus iserreḥ-ed i Barabas, jelkḍen Sidna Ɛisa, dɣa yefka lameṛ a t-semmṛen ɣef wumidag.</li>
      <li>Iɛsekṛiyen n lḥakem ssufɣen Sidna Ɛisa ɣer ufrag, snejmaɛen-d akk iɛsekṛiyen nniḍen, zzin-as.</li>
      <li>Imiren kksen-as llebsa-s, sselsen as abeṛnus azeggaɣ.</li>
      <li>Zḍan taɛeṣṣabt s isennanen, ssersen-as-ț ɣef wuqeṛṛuy is, fkan-as aɣanim ɣer ufus-is ayeffus. Bdan kennun zdat-es, stehzayen fell-as, qqaṛen-as : Sslam fell-ak ay agellid n wat Isṛail !</li>
      <li>Ssusufen-t, kkaten-t s uɣanim ɣer uqeṛṛuy.</li>
      <li>Mi ɛyan deg ustehzi fell-as, kksen-as abeṛnus-nni, rran-as llebsa-s, wwin-t a t-semmṛen ɣef wumidag.</li>
      <li>Mi d-ffɣen, mlalen-d yiwen wergaz n tmurt n Qiṛwan, isem-is semɛun, ḥettmen fell-as ad ibbib amidag n Sidna Ɛisa.</li>
      <li>Mi wwḍen ɣer yiwen wemkan ițțusemman Gulguṭa, yeɛni « amkan n uxecxuc »,</li>
      <li>fkan-as ad isew ccṛab i deg yexleḍ yiwen uɛeqqaṛ aṛzagan. Iɛreḍ-it, yugi a t-isew.</li>
      <li>Mi t-semmṛen ɣef wumidag, gren tasɣaṛt iwakken ad feṛqen llebsa-ines, s wakka, yedṛa-d wayen yuran di tira iqedsen : feṛqen llebsa-w, gren tasɣart ɣef wuqenduṛ-iw.</li>
      <li>Imiren qqimen a t-ɛassen,</li>
      <li>ɛelqen sennig uqeṛṛuy-is talwiḥt i ɣef uran sebba n lmut-is : Wagi d Ɛisa, agellid n wat Isṛail</li>
      <li>Țwasemmṛen yid-es sin imcumen, yiwen ɣer uyeffus-is wayeḍ ɣer uẓelmaḍ-is.</li>
      <li>Imsebriden țhuzzun iqeṛṛay-nsen reggmen-t,</li>
      <li>qqaṛen-as : A win ițhuddun lǧameɛ iqedsen ibennu-t di tlata wussan, sellek iman-ik ma d Mmi-s n Ṛebbi i telliḍ, ers-ed seg umidag-nni.</li>
      <li>Imeqqranen n lmuqedmin, lɛulama n ccariɛa akk-d imeqqranen n wegdud, stehzayen fell-as ula d nutni, qqaṛen :</li>
      <li>Isellek wiyaḍ ur yezmir ad isellek iman-is ! Ma d agellid n wat Isṛail i gella, a d-yers ihi seg umidag iwakken a namen yis.</li>
      <li>Yețkel ɣef Ṛebbi, a t-isellek ihi Ṛebbi ma iḥemmel-it aaxaṭer yenna-d : « Nekk d Mmi-s n Ṛebbi.»</li>
      <li>Ula d imcumen-nni ițțusemmṛen yid-es, regmen-t am wiyaḍ.</li>
      <li>Deg uzzal si tnac armi d țlata n tmeddit, yeɣli-d ṭṭlam ɣef ddunit meṛṛa.</li>
      <li>Ɣef leǧwahi n țlata, Sidna Ɛisa iɛeggeḍ s tɛibṛanit, yenna :  Ilya, Ilya, lama sabaqtani ? Yeɛni : Illu-iw, Illu-iw, acuɣeṛ i yi-teǧǧiḍ ?</li>
      <li>Kra seg wid iḥedṛen dinna slan d acu i genna, qqaṛen : Issawal i nnbi Ilyas !</li>
      <li>Imiren kan yiwen seg-sen yuzzel yeddem-ed ameččim n taḍuṭ isselxes-it di lxell, icudd-it ɣef wuɣanim yefka-yas ad isew.</li>
      <li>Wiyaḍ qqaṛen : Anfet-as, a nwali ma yella a d-yas nnbi Ilyas a t-isellek.</li>
      <li>Sidna Ɛisa iɛeggeḍ tikkelt nniḍen, dɣa iffeɣ-it Ṛṛuḥ.</li>
      <li>Imiren kan abeḥnuq n leḥjab yellan di lǧameɛ iqedsen icerreg ɣef sin, seg ixef ufella armi d akessar. Lqaɛa tergagi, idurar ceqqeqen,</li>
      <li>iẓekwan ldin, aṭas seg iḥeqqiyen yemmuten, ḥyan-d</li>
      <li>ffɣen-d seg iẓekwan-nsen. Mi d-iḥya Sidna Ɛisa, kecmen ɣer temdint iqedsen, aṭas n lɣaci iwumi i d-ḍehṛen.</li>
      <li>Lḥakem yellan d aqeṛṛuy ɣef meyya iɛsekṛiyen akk-d iɛsekṛiyen iɛussen Sidna Ɛisa, tekcem-iten tugdi mi walan tergagi lqaɛa d wayen akk yedṛan, nnan : Ț-țideț, argaz-agi d Mmi-s n Ṛebbi !</li>
      <li>Aṭas n tilawin i gellan dinna, țmuqulent-ed si lebɛid. Ț-țiggad-nni i d-yeddan d Sidna Ɛisa si tmurt n Jlili iwakken a s-qedcent.</li>
      <li>Tella gar-asent Meryem tamagdalit, Meryem yemma-s n Yeɛqub d Yusef akk-d d yemma-s n warraw n Zabadi.</li>
      <li>Mi d-tewweḍ tmeddit, yusa-d yiwen umeṛkanti n temdint n Arimati isem-is Yusef, ula d nețța d anelmad n Sidna Ɛisa.</li>
      <li>Iṛuḥ ɣer Bilaṭus yessuter-as lǧețța n Sidna Ɛisa. Bilaṭus yumeṛ ad a s-ț-id-fken.</li>
      <li>Yusef yeddem-ed lǧețța n Sidna Ɛisa, ițțel-iț di lekfen zeddigen,</li>
      <li>issers-iț deg yiwen uẓekka ajdid, i geɣza deg uzṛu. Yeɣleq tawwurt n uẓekka-nni s weblaḍ ameqqran, dɣa iṛuḥ.</li>
      <li>Meryem tamagdalit akk-d Meryem-nni nniḍen, llant dinna qqiment, qublent-ed aẓekka.</li>
      <li>Azekka nni yellan d ass n uheggi n wass n westeɛfu, imeqqranen n lmuqedmin d ifariziyen ṛuḥen ɣer Bilaṭus</li>
      <li>nnan-as : A Sidi, necfa ɣef wayen i d-yenna ukeddab-agi asmi yedder, yenna-d : Ad ɛeddin tlata wussan, a d-ḥyuɣ si ger lmegtin .</li>
      <li>Ceggeɛ iɛessasen ad ɛassen aẓekka-s alamma d ass wis tlata, iwakken inelmaden-is ur d-țasen ara ad akren lǧețța-s, ad inin i wegdud : « iḥya-d si ger lmegtin. » Neɣ m'ulac lekdeb-agi aneggaru ad yaggar amezwaru.</li>
      <li>Bilaṭus yenna-yasen-d : Awit iɛessasen, ṛuḥet ɛasset-ț akken tebɣam.</li>
      <li>Ṛuḥen, zemmemen aken ilaq tawwurt n uẓekka-nni s weblaḍ, imiren sbedden taɛessast zdat-es.</li>
    </ol>

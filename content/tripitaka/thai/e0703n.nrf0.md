<h2>มหาวํสปาฬิ</h2>
<h1>ปถมปริเจฺฉท</h1>
<h3>มหิยงฺคณาคมนํ</h3>
<ol>
<li>
นมสฺสิตฺวาน  
สมฺพุทฺธํ, สุสุทฺธํ สุทฺธวํสชํ;  
  
มหาวํสํ ปวกฺขามิ, นานานูนาธิการิกํฯ  
</li>
  
<li>
โปราเณหิ กโต’เปโส, อติวิตฺถาริโต กฺวจิ;  
  
อตีว กฺวจิ สํขิโตฺต, อเนกปุนรุตฺตโกฯ  
</li>
  
<li>
วชฺชิตํ เตหิ โทเสหิ, สุขคฺคหณธารณํ;  
  
ปสาทสํเวคกรํ, สุติโต จ อุปาคตํฯ  
</li>
  
<li>
ปสาทชนเก ฐาเน, ตถา สํเวคการเก;  
  
ชนยโนฺต ปสาทญฺจ, สํเวคญฺจ สุณาถ ตํฯ  
</li>
  
<li>
ทีปงฺกรญฺหิ สมฺพุทฺธํ, ปสฺสิตฺวา โน ชิโน ปุรา;  
  
โลกํ ทุกฺขา ปโมเจตุํ, โพธาย ปณิธิํ อกาฯ  
</li>
  
<li>
ตโต ตเญฺจว สมฺพุทฺธํ, โกณฺฑญฺญํ มงฺคลํ มุนิํ;  
  
สุมนํ เรวตํ พุทฺธํ, โสภิตญฺจ มหามุนิํฯ  
</li>
  
<li>
อโนมทสฺสิํ สมฺพุทฺธํ, ปทุมํ นารทํ ชินํ;  
  
ปทุมุตฺตรสมฺพุทฺธํ, สุเมธญฺจ ตถาคตํฯ  
</li>
  
<li>
สุชาตํ ปิยทสฺสิญฺจ, อตฺถทสฺสิญฺจ นายกํ;  
  
ธมฺมทสฺสิญฺจ สิทฺธตฺถํ, ติสฺสํ ผุสฺสชินํ ตถาฯ  
</li>
  
<li>
วิปสฺสิํ สิขีสมฺพุทฺธํ, สมฺพุทฺธํ เวสฺสภุํ วิภุํ;  
  
กกุสนฺธญฺจ สมฺพุทฺธํ, โกณาคมนเมว จฯ  
</li>
  
<li>
กสฺสปํ สุคตญฺจ’เม, สมฺพุเทฺธ จตุวีสติ;  
  
อาราเธตฺวา มหาวีโร, เตหิ โพธาย พฺยากโตฯ  
</li>
  
<li>
ปุเรตฺวา  
ปารมี สพฺพา, ปตฺวา สโมฺพธิมุตฺตมํ;  
  
อุตฺตโม โคตโม พุโทฺธ, สเตฺต ทุกฺขา ปโมจยิฯ  
</li>
  
<li>
มคเธสุ รุเวลายํ, โพธิมูเล มหามุนิ;  
  
วิสาขปุณฺณมายํ โส, ปโตฺต สโมฺพธิมุตฺตมํฯ  
</li>
  
<li>
สตฺตาหานิ ตหิํ สตฺต, โส วิมุตฺติสุขํ ปรํ;  
  
วินฺทนฺตํ มธุรตฺตญฺจ, ทสฺสยโนฺต วสี วสิฯ  
</li>
  
<li>
ตโต พาราณสิํ คนฺตฺวา, ธมฺมจกฺกํ ปคตฺตยิ;  
  
ตตฺถ วสฺส วสโนฺต’ว, สฎฺฐิํ อรหตํ อกาฯ  
</li>
  
<li>
เต ธมฺมเทสนตฺถาย, วิสฺสเชฺชตฺวาน ภิกฺขโว;  
  
วิเนตฺวา จ ตโต ติํส-สหา เย ภทฺทวคฺคิเยฯ  
</li>
  
<li>
สหสฺสชฎิเล นาโถ, วิเนตุํ กสฺสปาทิเก;  
  
เหมเนฺต อุรุเวลายํ, วสิเต ปริปาจยํฯ  
</li>
  
<li>
อุรุเวลกสฺสปสฺส, มหายเญฺญ อุปฎฺฐิเต;  
  
ตสฺส’ตฺตโน นาคมเน, อิจฺฉาจารํ วิชานิยฯ  
</li>
  
<li>
อุตฺตรกุรุโต ภิกฺขํ, อาหริตฺวา ริมทฺทโน;  
  
อโนตตฺตทเห ภุตฺวา, สายนฺหสมเย สยํฯ  
</li>
  
<li>
โพธิโต นวเม มาเส, ผุสฺสปุณฺณมิยํ ชิโน;  
  
ลงฺกาทีปํ วิโสเธตุํ, ลงฺกาทีปมุปาคมิฯ  
</li>
  
<li>
สาสนุโชฺชตนํ ฐานํ, ลํกา ญาตา ชิเนน หิ;  
  
ยกฺขปุณฺณาย ลงฺกาย, ยกฺขา นิพฺพา สิยาติ จฯ  
</li>
  
<li>
ญาโตว ลงฺกามชฺฌมฺหิ, คงฺคาภีเร มโนรเม;  
  
ติโยชนาย เต รเมฺม, เอกโยชนวิตฺถเตฯ  
</li>
  
<li>
มหานาควนุยฺยาเน, ยกฺขสงฺคามภูมิยา;  
  
ลงฺกาทีปฎฺฐยกฺขานํ, มหายกฺขสมาคโมฯ  
</li>
  
<li>
อุปาคโต ตํ สุคโต, มหายกฺขสมาคมํ;  
  
สมาคมสฺส มชฺฌมฺหิ, ตตฺถ เตสํ สิโรปริฯ  
</li>
  
<li>
มหิยงฺคณถูปสฺส, ฐาเน เวหายสํ ชิโน;  
  
วุฎฺฐิวาตนฺธกาเรสิ, เตสํ สํเวชนํ อกาฯ  
</li>
  
<li>
เต ภยฎฺฐา’ภยํ ยกฺขา, อยาจุํ อภยํ ชินํ;  
  
ชิโน อภยโท อาห, ยเกฺข เต’ติ ภยทฺทีเตฯ  
</li>
  
<li>
ยกฺขา  
ภยํ โว ทุกฺขญฺจ, หริสฺสามิ อิทํ อหํ;  
  
ตุเมฺห นิสชฺชฐานํ เม, สมคฺคา เทถ โน อิธฯ  
</li>
  
<li>
อาหุ เต สุคตํ ยกฺขา, เทม มาริส เต อิธ;  
  
สเพฺพปิ สกลํ ทีปํ, เทหิ โน อภยํ ตุวํฯ  
</li>
  
<li>
ภยํ สีตํ ตมํ เตสํ, หนฺตฺวา ตํ ทินฺนภูมิยํ;  
  
จมฺมกฺขนฺธํ อตฺถริตฺวา, ตตฺถา’ สีโน ชิโน ตโตฯ  
</li>
  
<li>
จมฺมกฺขณฺฑํ ปสาเรสิ, อาทิตฺตํ ตํ สมนฺตโต;  
  
ฆมฺมาภิภูตา เต ภีตา, ฐิตา อเนฺต สมนฺตโตฯ  
</li>
  
<li>
คิริทีปํ ตโต นาโถ, รมฺมํ เตสํ อิธา’นยิ;  
  
เตสุ ตตฺถ ปวิเฎฺฐสุ, ยถาฐาเน ฐเปสิ จฯ  
</li>
  
<li>
นาโถ ตํ สํขิปิ ธมฺมํ, ตทา เทวา สมาคมุํ;  
  
ตสฺมิํ สมาคเม เตสํ, สตฺถา ธมฺม มเทสยิฯ  
</li>
  
<li>
เนเกสํ ปาณโกฎีนํ, ธมฺมาภิสมโย อหุ;  
  
สรเณสุ จ สีเลสุ, ฐิตา อาสุํ อสํขิยาฯ  
</li>
  
<li>
โสตาปตฺติผลํ ปตฺวา, เสเล สุมนกูฎเก;  
  
มหาสุมนเทวิโนฺท, ปูชิยํ ยาจิ ปูชิยํฯ  
</li>
  
<li>
สิรํ ปรามสิตฺวาน, นีลามลสิโรรุโห;  
  
ปาณิมเตฺตอทา เกเส, ตสฺส ปาณ หิโต ชิโนฯ  
</li>
  
<li>
โส ตํ สุวณฺณจโงฺกฎ-วเรนาทาย สตฺถุโน;  
  
นิสินฺนฎฺฐานรจิเต, นานารตนสญฺจเยฯ  
</li>
  
<li>
สพฺพโต สตฺตรตเน, เต ฐเปตฺวา สิโรรุเห;  
  
โส อินฺทนีลถูเปน, ปิทเหสิ นมสฺสิ จฯ  
</li>
  
<li>
ปรินิพฺพุตมฺหิ สมฺพุเทฺธ, จิตกโต จ อิทฺธิยา;  
  
อาทาย ชินคีวฎฺฐิํ, เถโร สรภูนามโกฯ  
</li>
  
<li>
เถรสฺส สาริปุตฺตสฺส, สิโสฺส อานีย เจติเย;  
  
ตสฺมิํเยว ฐเปตฺวาน, ภิกฺขูหิ ปริวาริโตฯ  
</li>
  
<li>
ฉาทาเปตฺวา เมทวณฺณ-ปาสาเณหิ มหิทฺธิโก;  
  
ถูปํ ทฺวาทสหตฺถุจฺจํ, การาเปตฺวาน ปกฺกมิฯ  
</li>
  
<li>
เทวานํปิยติสฺสสฺส, รโญฺญ ภาตุกุธารโก;  
  
อุทฺธจูฬาภโย นาม, ทิสฺวา เจติย มพฺภุตํฯ  
</li>
  
<li>
ตํ  
ฉาทยิตฺวา กาเรสิ, ติํสหตฺถุจฺจ เจติยํ;  
  
มทฺทโนฺต ทมิเฬ ราชา, ตตฺรโฎฺฐ ทุกฺขคามณิฯ  
</li>
  
<li>
อสีติหตฺถํ กาเรสิ, ตสฺส กญฺชุกเจติยํ;  
  
มหิยงฺคณถูโปย-เมโส เอวํ ปติฎฺฐิโตฯ  
</li>
  
<li>
เอวํ ทีปมิมํกตฺวา, มนุสฺสารหมิสฺสโร;  
  
อุรุเวลมคา ธีโร, อุรุวีร ปรกฺกโมติฯ  
</li>
  
มหิยงฺคณาคมนํ นิฎฺฐิตํฯ  
</li>

มหตี อนุกมฺปาโน, กตา นาถ ตยา อยํ;

<h1>เอกวีสติม ปริเจฺฉท</h1>
<h3>วญฺจ ราชโก</h3>
<ol>
<li>
อุตฺติยสฺส  
กนิโฎฺฐ ตุ, มหาสิโว ตทจฺจเย;  
  
ทส วสฺสานิ กาเรสิ, รชฺชํ สุชน เสวโกฯ  
</li>
  
<li>
ภทฺทสาลมฺหิ โส เถเร, ปสีทิตฺวา มโนรมํ;  
  
กาเรสิ ปุริมายนฺตุ, วิหารํ นครงฺคณํฯ  
</li>
  
<li>
มหาสิวกนิโฎฺฐ ตุ, สุรติโสฺส ตทจฺจเย;  
  
ทสวสฺสานิ กาเรสิ, รชฺชํ ปุเญฺญสุ สาทโรฯ  
</li>
  
<li>
ทกฺขิณาย ทิสายํ โส, วิหารํ นครํ คณํ;  
  
ปุริมาย หตฺถิกฺขนฺธญฺจ, โคกณฺณคิริเมว จฯ  
</li>
  
<li>
วงฺคุตฺตเร ปพฺพตมฺหิ, ปาจิน ปพฺพตวฺหยํ;  
  
รเหณกสมีปมฺหิ, ตถา โกฬมฺพกาฬกํฯ  
</li>
  
<li>
อริฎฺฐปาเท มํ คุลกํ, ปุริมาย’จฺฉ คลฺลกํ;  
  
คิริเนลปนายกณฺฑํ, นครสฺสุตฺถราย ตุฯ  
</li>
  
<li>
ปญฺจสตา  
เนว มาทิ-วิหาเร ปุถวี ปติ;  
  
คงฺคาย โอรปารญฺหิ, ลํกาทีเป ตหิํ ตหิํฯ  
</li>
  
<li>
ปุเร รชฺชา จ รเชฺช จ, สฎฺฐีวสฺสานิ สาธุกํ;  
  
กาเรสิ รเมฺม ธเมฺมน, รตนตฺตย คารโวฯ  
</li>
  
<li>
สุวณฺณปิณฺฑติโสฺสติ, นามํ รชฺชา ตสฺสา อหุ;  
  
สุรติโสฺสติ นามนฺตุ, ตสฺสา’ปุรชฺชปตฺติยาฯ  
</li>
  
<li>
อสฺสนาวิกปุตฺตา เทฺว, ทมิฬา เสนคุตฺติกา;  
  
สุรติสฺสมหีปาลํ, ตํ คเหตฺวา มหพฺพลาฯ  
</li>
  
<li>
ทุเว วีสติวสฺสานิ, รชฺชํ ธเมฺมน การยุํ;  
  
เต คเหตฺวา อเสโล ตุ, มุฎสิวสฺส อตฺรโชฯ  
</li>
  
<li>
โสทริยานํ ภาตูนํ, นวโม ภตุโก ตโต;  
  
อนุราธปุเร รชฺชํ, ทสวสฺสานิ การยิฯ  
</li>
  
<li>
โจฬรฎฺฐา อิธ คมฺม, รชฺชตฺถํ อุชุชาติโก;  
  
เอฬาโร นาม ทมิโฬ, คเหตฺวา’เสลภูปติํฯ  
</li>
  
<li>
วสฺสานิ จตฺตารีสญฺจ, จตฺตาริ จ อการยิ;  
  
รชฺชํ โวหารสมเย, มชฺฌโตฺต มิตฺตสตฺตุสุฯ  
</li>
  
<li>
สยนสฺส สิโรปเสฺส, ฆณฺฎํ สุทีฆโยตฺตกํ;  
  
ลมฺพาเปสิ วีราเวตุํ, อิจฺฉเนฺตหิ วินิจฺฉยํฯ  
</li>
  
<li>
เอโก ปุโตฺต จ ธีตา จ, อเหสุํ ตสฺส ราชิโน;  
  
รเถน ติสฺสวาปิํ โส, คจฺฉโนฺต ภูมิปาลโชฯ  
</li>
  
<li>
ตรุณํ วจฺฉกํ มเคฺค, นิปนฺนํ สหเธนุกํ;  
  
คีวํ อกฺกมฺมจเกฺกน, อสญฺจิจฺจ อฆาตยิฯ  
</li>
  
<li>
คนฺตฺวาน เธนุฆณฺฎํ ตํ, ฆเฎฺฎสิ ฆฎฺฎิตาสยา;  
  
ราชา เตเนว จเกฺกน, สีสํ ปุตฺตสฺส เฉทยิฯ  
</li>
  
<li>
ทิชโปตํ ตาลรุเกฺข, เอโก สโปฺป อภกฺขยิ;  
  
ตโมฺปตมาตา สกุณี, คนฺตฺวา ฆณฺฎมฆฎฺฎยิฯ  
</li>
  
<li>
อาณาเปตฺวาน ตํ ราชา, กุจฺฉิํ ตสฺส วิทาฬิย;  
  
โปตํ ตํ นีหราเปตฺวา, ตาเล สปฺปมสปฺปยิฯ  
</li>
  
<li>
รตนคฺคสฺส  
รตน-ตฺตยสฺส คุณสารตํ;  
  
อชานโนฺตปิ โส ราชา, จาริตฺตมนุปาลยํฯ  
</li>
  
<li>
เจติยทพฺพตํ คนฺตฺวา, ภิกฺขุสงฺฆํ ปวาริย;  
  
อาคจฺฉโนฺต รถคโต, รถสฺส ยุคโกฎิยาฯ  
</li>
  
<li>
อกาสิ ชินถูปสฺส,  
  
เอกเทสสฺส ภญฺชนํ;  
  
อมจฺจา ‘‘เทว ถูโปโน,  
  
ตยา ภิโนฺน’ติ อาหุ ตํฯ  
</li>
  
<li>
อสญฺจิจฺจ กเต‘เป’ส, ราชา โอรุยฺห สนฺทนา;  
  
‘‘จเกฺกน มม สีสมฺปิ, ฉินฺทถา’ติ ปเถสยิฯ  
</li>
  
<li>
‘‘ปรหิํสํ มหาราช, สตฺถา โน เนว อิจฺฉติ;  
  
ถูปํ ปากติกํ กตฺวา, ขมาเปหี’’ติ อาหุ ตํฯ  
</li>
  
<li>
เต ฐเปตุํ ปญฺจทส, ปาสาเณ ปติเต ตหิํ;  
  
กหาปณ สหสฺสานิ, อทา ปญฺจทเส วโสฯ  
</li>
  
<li>
อกา มหลฺลิกา วีหิํ, โส เสตุํ อาตเป ขิปิ;  
  
เทโว อกาเล วสฺสิตฺวา, ตสฺสา วิหิํ อเตมยิฯ  
</li>
  
<li>
วีหิํ คเหตฺวา คนฺตฺวา สา, ฆณฺฎํ ตํ สมฆฎฺฎยิ;  
  
อกาลวสฺสํ สุตฺวา ตํ, วิสฺสเชฺชตฺวา ตมตฺติกํฯ  
</li>
  
<li>
ราชา ธมฺมมฺหิ วตฺตโนฺต, ‘‘กาเล วสฺสํ ลเภ’’อิติ;  
  
ตสฺสา วินิจฺฉยตฺถาย, อุปวาสํ นิปชฺชิ โสฯ  
</li>
  
<li>
พลิคฺคาหี เทวปุโตฺต, รโญฺญ เตเชน โอตฺถโฎ;  
  
คนฺตฺวา จาตุมหาราช-สนฺติกํ ตํ นิเวทยิฯ  
</li>
  
<li>
เต ตมาทาย คนฺตฺวาน, สกฺกสฺส ปฎิเวทยุํ;  
  
สโกฺก ปชฺชุนฺนมาหุย, กาเล วสฺสํ อุปาทิยิฯ  
</li>
  
<li>
พลิคฺคาหี เทวปุโตฺต, ราชิโน ตํ นิเวทยิ;  
  
ตโตปฺปภุติ ตํ รเชฺช, ทิวา เทโว น วสฺสถฯ  
</li>
  
<li>
รตฺติํ เทโว’นุ สตฺตาหํ, วสฺสิยามมฺหิ มชฺฌิเม;  
  
ปุณฺณาน’เหสุํ สพฺพตฺถ, ขุทฺทกา วาฎกานิปิฯ  
</li>
  
<li>
อคติคมนโทโส  
มุตฺตมเตฺตน เอโส,  
  
อนุปหต กุทิฎฺฐิปีทิสิํ ปาปุณี’ทฺธิํ;  
  
อคติคมนโทสํ สุทฺธทิฎฺฐิสมาโน,  
  
กถมิธหิ มนุโสฺส พุทฺธิมา โน ชเหยฺยาติฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
ปญฺจราชโก นาม  
</li>
  
เอกวีสติโม ปริเจฺฉโทฯ  
</li>

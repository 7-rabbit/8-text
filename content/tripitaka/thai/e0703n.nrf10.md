<h1>เอกาทสม ปริเจฺฉท</h1>
<h3>เทวานํปิยติสฺสาภิเสโก</h3>
<ol>
<li>
ตสฺสจฺจเย  
ตสฺส สุโต, มุฎสิโว’ติ วิสฺสุโต;  
  
สุวณฺณปาลิยา ปุโตฺต, ปโตฺต รชฺชมนากุลํฯ  
</li>
  
<li>
มหาเมฆวนุยฺยานํ, นามนุคคุโณทิตํ;  
  
ผลปุปฺผตรุเปตํ, โส ราชา การยิ สุภํฯ  
</li>
  
<li>
อุยฺยานฐานคฺคหเณ, มหาเมโฆ อกาลโช;  
  
ปาวสฺสิเตน อุยฺยานํ, มหาเมฆวนํ อหุฯ  
</li>
  
<li>
สฎฺฐิวสฺสานิ มุฎสิโว, ราชา ราชฺชมการยิ;  
  
อนุราเธ ปุรวเร, ลํกา ภุวทเน สุเภฯ  
</li>
  
<li>
ตสฺส ปุตฺตา ทสา’เหสุํ, อญฺญมญฺญหิเตสิโน;  
  
ทุเว ธีตา จา’นุกุลา, กุลานุจฺฉวิกา อหูฯ  
</li>
  
<li>
เทวานํ ปิยติโสฺสติ, วิสฺสุโต ทุติโย สุโต;  
  
เตสุ ภาตูสุ สเพฺพสุ, ปุญฺญาปญฺญาธิโก อหุฯ  
</li>
  
<li>
เทวานํ ปิยติโสฺส โส, ราชา’สิ ปิตุอจฺจเย;  
  
ตสฺสา’ภิเสเกน สมํ, พหูน’จฺฉริยาน’หุํฯ  
</li>
  
<li>
ลํกาทีปมฺหิ สกเล, นิธโย รตนานิ จ;  
  
อโนฺตฐิตานิ อุคฺคนฺตฺวา, ปถวีตลมารุหุํฯ  
</li>
  
<li>
ลํกาทีปสมีปมฺหิ, ภินฺนนาวาคตานิ จ;  
  
ตตฺรชาตานิ จ ถลํ, รตนานิ สมารุหุํฯ  
</li>
  
<li>
ฉาตปพฺพตปาทมฺหิ, ติโสฺส จ เวฬุยฎฺฐิโย;  
  
ชาตารถปโตเทน, สมานา ปริมาณโตฯ  
</li>
  
<li>
ตาสุ เอกาลตายฎฺฐิ, รชตา’ภา ตหิํ ลตา;  
  
สุวณฺณาวณฺณารุจิรา, ทิสฺสเนฺตตา มโนรมาฯ  
</li>
  
<li>
เอกา กุสุมา ยฎฺฐีตุ, กุสุมานิ ตหิํ ปน;  
  
นานานิ นานาวณฺณานิ, ทิสฺสเนฺต’ติ ผุฎานิ จฯ  
</li>
  
<li>
เอกาสกุณยฎฺฐิตุ  
, ตหิํ ปกฺขิมิคา พหู;  
  
นานา จ นานาวณฺณา จ, สชีวา วิย ทิสฺสเรฯ  
</li>
  
<li>
ภยคชรถามลกา, วลยงฺคุลิเวฐกา;  
  
กกุธผลาปากติกา, อิเจฺจตา อฎฺฐชาติโยฯ  
</li>
  
<li>
มุตฺตาสมุทา อุคฺคนฺตฺวา, ตีเร วฎฺฎิ วิย ฐิตา;  
  
เทวานํ ปิยติสฺสสฺส, สพฺพํ ปุญฺญวิชมฺภิตํฯ  
</li>
  
<li>
อินฺทนีลํ เวฬุริยํ, โลหิตงฺกมณิธิ’เม;  
  
รชตานิ ปเน’ตานิ, มุตฺตา ตา ตาวยฎฺฐิโยฯ  
</li>
  
<li>
สตฺตหพฺภนฺตเรเยว, รโญฺญ สนฺติกมาหรุํ;  
  
ตานิ ทิสฺวา ปตีโต โส, ราชา อิติ วิจินฺตยิฯ  
</li>
  
<li>
รตนานิ อนคฺฆานิ, ธมฺมาโสโก อิมานิเม;  
  
สหาโย’รหเตนา’โญฺญ, ตสฺส ทสฺสํ อิมาน’โตฯ  
</li>
  
<li>
เทวานํ ปิยติโสฺส จ, ธมฺมาโสโก จ เต อิเม;  
  
เทฺว อทิฎฺฐสหายาหิ, จิรปฺปภุติ ภูปติฯ  
</li>
  
<li>
ภาคิเนยฺยํ มหาริฎฺฐํ, อมจฺจํ ปมุขํ ตโต;  
  
ทิชํ อมจฺจํ คณตํ, ราชา เต จตุโร ชเนฯ  
</li>
  
<li>
ทูโต กตฺวาน ปาเหสิ, พโลฆปริวาริเต;  
  
คาหา เปตฺวา อนคฺฆานิ, รตนานิ อิมานิ โสฯ  
</li>
  
<li>
มณิชาตี จ ติโสฺส ตา, ติโสฺส จ รถยฎฺฐิโย;  
  
สงฺขญฺจ ทกฺขิณาวตฺตํ, มุตฺตา ชาตี จ อฎฺฐ ตาฯ  
</li>
  
<li>
อารุยฺห ชมฺพุโกลมฺหิ, นาวํ สตฺตทิเนน เต;  
  
สุเขน ติตฺถํ ลทฺธาน, สตฺตาเหน ตโต ปุนฯ  
</li>
  
<li>
ปาฎลิปุตฺตํ คนฺตฺวาน, ธมฺมาโสกสฺส ราชิโน;  
  
อทํสุ ปณฺณากาเรเต, ทิสฺวา ตานิ ปสีทิ โสฯ  
</li>
  
<li>
รตนานิ’ทิสาเนตฺถ, นตฺถิ เม อิติ จินฺติย;  
  
อทา เสนาปติฎฺฐานํ, ตุโฎฺฐ’ริฎฺฐสฺส ภูปติฯ  
</li>
  
<li>
โปโรหิจฺจพฺราหฺมณสฺส, ทณฺฑนายตํ ปน;  
  
อทาสิ ตสฺสา’มจฺจสฺส, เสฎฺฐิตฺตํ คณกสฺส ตุฯ  
</li>
  
<li>
เตสํ อนปฺปเก โภเค, ทตฺวา วาสฆรานิ จ;  
  
มหามเจฺจหิ มเนฺตโนฺต, ปสฺสิตฺวา ปฎิปาภตํฯ  
</li>
  
<li>
วาลพีชนี  
มุณฺหิสํ, ขคฺคํ ฉตฺตญฺจ ปาทุกํ;  
  
โมฬิํ วตํ สปามงฺคํ, ภิงฺการํ หริจนฺทนํฯ  
</li>
  
<li>
อโธวิมํ วตฺถโกฎิํ, มหคฺฆํ หตฺถปุญฺฉนิํ;  
  
นาคา’หฎํ อญฺชนญฺจ, อรุณาภญฺจ มตฺติกํฯ  
</li>
  
<li>
อโนตโตฺตทกเญฺจว, คงฺคาสลิลเมว จ;  
  
สงฺขญฺจ นนฺทิยาวฎฺฎํ, วฑฺฒมานํ กุมาริกํฯ  
</li>
  
<li>
เหมโภชนกณฺฑญฺจ, สิวิกญฺจ มหารหํ;  
  
หรีฎกํ อามลกํ, มหคฺฆํ อมโตสธํฯ  
</li>
  
<li>
สุกาหฎานํ สาลีนํ, สฎฺฐิวาหสตานิ จ;  
  
อภิเสโก ปกรณํ, ปริวารวิเสสิตํฯ  
</li>
  
<li>
ทตฺวา กาเล ภหายสฺส, ปณฺณากาเร นริสฺสโร;  
  
ทูเต ปาเหสิสทฺธมฺม-ปณฺณาการมิมมฺปิจฯ  
</li>
  
<li>
อหํ พุทฺธญฺจ ธมฺมญฺจ, สงฺฆญฺจ สรณํ คโต;  
  
อุปาสกตฺตํ เทเสสิํ, สกฺยปุตฺตสฺส สาสเนฯ  
</li>
  
<li>
ตฺวมฺปิมานิ รตนานิ, อุตฺตมานิ นรุตฺตม;  
  
จิตฺตํ ปสาทยิตฺวาน, สทฺธาย สรณํ วชฯ  
</li>
  
<li>
‘‘กโรถ เม สหายสฺส, อภิเสกํ ปุโน’’ อิติ;  
  
วตฺวา สหยา’มเจฺจเต, สกฺกริตฺวา จ เปสยิฯ  
</li>
  
<li>
ปญฺจมาเส วสิตฺวาน, เต มจฺจา’ติวสกฺกตา;  
  
เวสาขสุกฺขปกฺขาทิ-ทิเน ทูตา วินิคฺคตาฯ  
</li>
  
<li>
ตามลิตฺติยมารุยฺห, นาวํ เตชมฺพุโกลเก;  
  
โอรุยฺห กูปํ ปสฺสิํสุ, ปตฺวา ทฺวาทสิยํ อิธฯ  
</li>
  
<li>
อทํสุ ปณฺณากาเร เต, ทูตา ลํกาทีปสฺสเต;  
  
เตสํ มหนฺตํ สกฺการํ, ลํกาปติ อการยิฯ  
</li>
  
<li>
เต มคฺคสิรมาสสฺส, อาทิจโนฺททเยชิเน;  
  
อภิสิตฺตญฺจ ลํกินฺทํ, อมจฺจา สามิภตฺติโนฯ  
</li>
  
<li>
ธมฺมาโสกสฺส วจนํ, วตฺวา สามิ หิเตรตา;  
  
ปุโนปิ อภิสิญฺจิํสุ, ลํกาหิตสุเขรตํฯ  
</li>
  
<li>
เวสาเข  
นรปติ ปุณฺณมายเมวํ;  
  
เทวานํ ปิยวจโน ปคูฬนาโม;  
  
ลํกายํ ปวีตตปีติ อุสฺสวายํ;  
  
อตฺตานํ ชนสุขโท’ภิเสจยี โสติฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
เทวานํปิยติสฺสาภิเสโกนาม  
</li>
  
เอกาทสโม ปริเจฺฉโทฯ  
</li>

<h1>อฎฺฐวีสติม ปริเจฺฉท</h1>
<h3>มหา ถูปสาธน ลาโภ</h3>
<ol>
<li>
ตโต สตสหสฺสํ โส, วิสฺสเชฺชตฺวา มหีปติ;  
  
การาเปสิ มหาโพธิ-ปูชํ สุฬารมุตฺตมํฯ  
</li>
  
<li>
ตโต ปุรํ ปวิสิโนฺต, ถูปฐาเน นิเวสิตํ;  
  
ปสฺสิตฺวาน สีลายูปํ, สริตฺวา ปุพฺพกํ สุติํฯ  
</li>
  
<li>
‘‘กาเรสฺสามิ มหาถูปํ’’, อิติ หโฎฺฐ มหาตลํ;  
  
อารุยฺห รตฺติํ ภุญฺชิตฺวา, สยิโต อิติ จินฺตยิฯ  
</li>
  
<li>
‘‘ทมิเฬ  
มทฺทมาเนน, โลโก’ยํ ปีฬิโต มยา;  
  
น สกฺกา พลิมุทฺธตฺตุํ, ตํ วชฺชิย พลิํ อหํฯ  
</li>
  
<li>
การยโนฺต มหาถูปํ, กตํ ธเมฺมน อิฎฺฐิกา;  
  
อุปฺปาเทสฺสามิ ‘‘อิเจฺจวํ, จินฺตยนฺตสฺส จินฺติตํฯ  
</li>
  
<li>
ฉตฺตมฺหิ เทวตาชานี, ตโต โกลาหลํ อหุ;  
  
เทเวสุ ญตฺวา ตํ สโกฺก, วิสฺสกมฺมาน มพฺรวิฯ  
</li>
  
<li>
‘‘อิฎฺฐกตฺตํ เจติยสฺส, ราชา จิเนฺตสิ คามณิ;  
  
คนฺตฺวา ปุรา โยชนมฺหิ, คมฺภีร นที ยนฺติเกฯ  
</li>
  
<li>
มาเปหิ อิฎฺฐิกา ตตฺตํ’, อิติ สเกฺก น ภาสิโต;  
  
วิสฺสกโมฺม อิธาคมฺม, มาเปสิ ตตฺถ อิฎฺฐิกาฯ  
</li>
  
<li>
ปภาเต ลุทฺทโก ตตฺถ, สุนเขหิ วนํ อคา;  
  
โวธารูเปน ทเสฺสสิ, ลุทฺทกํ ภุมฺมเทวตาฯ  
</li>
  
<li>
ลุทฺทโก ตํ’นุพนฺธโนฺต, คนฺตฺวา ทิสฺวาน อิฎฺฐกา;  
  
อนฺตรหิตาย โคธาย, อิติ จิธนฺตสิ โส ตหิํฯ  
</li>
  
<li>
‘‘กาเรตุ กาโม กิร โน, มหาถูปํ มหีปติ;  
  
อุปายนมิทํ ตสฺสํ’, อิติ คนฺตฺวา นิเวทยิฯ  
</li>
  
<li>
ตสฺส ตํ วจนํ สุตฺวา, ปิยํ ชนหิ ตปฺปิโย;  
  
ราชา กาเรสิ สกฺการํ, มหนฺตํ ตุฎฺฐมานโสฯ  
</li>
  
<li>
ปุราปุพฺพุตฺตเร เทเส, โยชนตฺตย มตฺถเก;  
  
อาจาร วิฎฺฐิคามมฺหิ, โสฬสกรีเส กเลฯ  
</li>
  
<li>
โสวณฺณ พีชานุ’ฎฺฐิํสุ, วิวิธานิ ปมาณโต;  
  
วิทตฺถุกฺกฎฺฐ มาณานิ, องฺคุลิมาณา นิเหฐโตฯ  
</li>
  
<li>
สุวณฺณ ปุณฺณํ ตํ ภูมิํ, ทิสฺวา สงฺคามวาสิกา;  
  
สุวณฺณ ปาหิํ อาทาย, คนฺตฺวา รโญฺญ นิเวทยุํฯ  
</li>
  
<li>
ปุราปาจิน ปสฺสมฺหิ, สตฺต โยชน มตฺถเก;  
  
คงฺคาปาเร ตมฺพปิเฎฺฐ, ตมฺพโลหํ สมุฎฺฐหิฯ  
</li>
  
<li>
ตงฺคามิกา ตมฺพโลห-พีชมาทาย ปาติยา;  
  
ราชาน มุปสงฺกมฺม, ตมตฺถญฺจ นิเวทยุํฯ  
</li>
  
<li>
ปุพฺพทกฺขิณเทสมฺหิ, ปุรโต จตุโยชเน;  
  
สุมน วาปิ คามมฺหิ, อุฎฺฐหิํสุ มณีพหูฯ  
</li>
  
<li>
อุปล  
กุรุวิเนฺทหิ, มิสฺสกาเนว คามิกา;  
  
อาทาย ปาติยาเอว, คนฺตฺวา รโญฺญ นิเวทยุํฯ  
</li>
  
<li>
ปุรโต ทกฺขิเณ ปเสฺส, อฎฺฐโยชนมตฺถเก;  
  
อมฺพฎฺฐ โกลเลณมฺหิ, รชตํ อุปปชฺชถฯ  
</li>
  
<li>
นคเร วาณิโช เอโก, อาทาย สกเฎ พหู;  
  
มลยา สิงฺคีเวราทิํ, อาเนตุํ มลยํ คโตฯ  
</li>
  
<li>
เลณสฺส อวิทูรมฺหิ, สกฎานิฎฺฐ ปาปิย;  
  
ปโตทาทารุนิจฺฉโนฺต, อารุโฬฺห ตํ มหีธรํฯ  
</li>
  
<li>
จาฎิปฺปมาณ ตํ ตตฺถ, ปกฺกภาเรน นามิ ตํ;  
  
ทิสฺวา ปน สลฎฺฐิญฺจ, ปาสาณฎฺฐญฺจ ตํ ผลํฯ  
</li>
  
<li>
วเณฺฎ ตํ วาสิยา เฉตฺวา, ทสฺสาม’คฺคนฺติ จินฺติย;  
  
กาลํ โฆเสสิ สทฺธาย, จตฺตาโร’นาสวาคมุํฯ  
</li>
  
<li>
หโฎฺฐ โส เต’ภิวาเทตฺวา, นิสีทาปิย สาทโร;  
  
วาสิยา วณฺฎสามนฺตา, ตจํ เฉตฺวา อปสฺสยํฯ  
</li>
  
<li>
มุญฺชิตฺวา’วาฎ ปุณฺณํ ตํ, ยูสํ ปเตฺตหี อาทิย;  
  
จตฺตาโร ยูส ปูเรเต, ปเตฺต เตสมทาสิโสฯ  
</li>
  
<li>
เต ตํ คเหตฺวา ปกฺกามุํ,  
  
กาลํ โฆเสสิ โส ปุน;  
  
อเญฺญ ขิณาสวา เถรา,  
  
จตฺตาโร ตตฺถ อาคมุํฯ  
</li>
  
<li>
เตสํ ปเตฺต คเหตฺวา โส, ปตฺตามิญฺชหิ ปูริย;  
  
อทาสิ เตสํ ปกฺกามุํ, ตโย เอโก นปกฺกมิฯ  
</li>
  
<li>
รชตํ ตสฺส ทเสฺสตุํ, โอโรหิตฺวา ตโต หิ โส;  
  
นิสชฺช เลณาสนฺนมฺหิ, ตามิญฺชา ปริภุญฺชถฯ  
</li>
  
<li>
เสสามิญฺชา วาณิโช’ปิ, ภุญฺชิตฺวา ยาวทตฺถกํ;  
  
ภณฺฑิกาย คเหตฺวาน, เสสเถรปทานุโคฯ  
</li>
  
<li>
อญฺชสา  
อิมินา ตฺวมฺปิ, คจฺฉ ทานี อุปาสก;  
  
คนฺตฺวาน เถรํ ปสฺสิตฺวา, เวยฺยาวจฺจ มหาสิ จฯ  
</li>
  
<li>
เถโร จ เลณทฺวาเรน, ตสฺส มคฺค อมาปยิ;  
  
เถรํ วนฺทิย โส เตน, คจฺฉโนฺต เลณมทฺทสฯ  
</li>
  
<li>
เลณ ทฺวารมฺหิ ฐตฺวาน, ปสฺสิตฺวา รชตมฺปิ โส;  
  
วาสิยา อาหนิตฺวาน, รชตติ วิชานิยฯ  
</li>
  
<li>
คเหเตฺวกํ สชฺฌุปิณฺฑํ, คนฺตฺวาน สกฎนฺติกํ;  
  
สกฎานิ ฐปาเปตฺวา, สชฺฌุปิณฺฑํ ตมาทิยฯ  
</li>
  
<li>
ลหุํ อนุราธ ปุรํ, อาคมฺม วรวาณิโช;  
  
ทเสฺสตฺวา รชตํ รโญฺญ, ตมตฺถมฺปิ นิเวทยิฯ  
</li>
  
<li>
ปุรโต ปจฺฉิเม ปเสฺส, ปญฺจโยชน มตฺถเก;  
  
อุรุเวลปฎฺฎเน มุตฺตา, มหามลก มตฺติโยฯ  
</li>
  
<li>
ปวาลนฺตริกา สทฺธิํ, สมุทฺทาถลโมกฺกมุํ;  
  
เกวฎฺฎา ตา สเมกฺขิตฺวา, ราสิํ กตฺวาน เอกโตฯ  
</li>
  
<li>
ปาติยา อานยิตฺวาน, มุตฺตา สห ปวาลกา;  
  
ราชาน มุปสงฺกมฺม, ตมตฺถมฺปิ นิเวทยุํฯ  
</li>
  
<li>
ปุรโต อุตฺตเร ปเสฺส, สตฺต โยชน มตฺถเก;  
  
เปฬิวาปิก คามสฺส, วาปิปกฺขนฺตกนฺทเรฯ  
</li>
  
<li>
ชายิํสุ วาลุกาปิเฎฺฐ, จตฺตาโร อุตฺตมา มณี;  
  
นีสท โปตปฺปมาณา, อุมฺมาปุปฺผนิภาสุภาฯ  
</li>
  
<li>
เต ทิสฺวา สุนโข ลุโทฺท, อคนฺตฺวา ราชสนฺติกํ;  
  
‘‘เอวรูปา มณีทิโฎฺฐ, มยา’’อิติ นิเวทยิฯ  
</li>
  
<li>
อิฎฺฐกาทีนิ เอตานิ มหาปุโญฺญ มหามติ;  
  
มหาถูปตฺถมุปฺปนฺนา-น’โสฺสสิ ตทเหว โสฯ  
</li>
  
<li>
ยถานุรูปํ สกฺการํ, เตสํ กตฺวา สุมานโต;  
  
เอเต วา รกฺขเก กตฺวา, สพฺพานิ อาหราปยิฯ  
</li>
  
<li>
เขทมฺปิ  
กายชมสยฺหมจินฺตยิตฺวา,  
  
ปุญฺญํ ปสนฺนมนโส ปจิต หิ เอวํ;  
  
สาเธติ สาธน สตานิ สุขา กรานิ,  
  
ตสฺมา ปสนฺนมนโส’ว กเรยฺย ปุญฺญนฺติฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
มหาถูปสาธนลาโภ นาม  
</li>
  
อฎฺฐวีสติโม ปริเจฺฉโทฯ  
</li>

<h1>ติสฎฺฐิม ปริเจฺฉท</h1>
<h3>เสนาปติวโธ</h3>
<ol>
<li>
นิชายุธทุติยสฺส  
, นิกฺขมนฺตสฺส ตสฺส หิ;  
  
ตํ ขณํ ปุรโต โก’ปิ, สงฺขสโทฺท สมุคฺคโตฯ  
</li>
  
<li>
ตโต เนกนิมิตฺตญฺญู, กุมาโร ตํ สุณิ ตฺว โส;  
  
นิปฺผชฺชิสฺสติ สงฺกโปฺป, ขิปฺปํ เยเวติ โมทวาฯ  
</li>
  
<li>
ตตฺถ ตตฺถ นิยุตฺตานํ, รกฺขกานมชานตํ;  
  
นิกฺขมิตฺวา ปุราวีต-ภโย สีหปรกฺกโมฯ  
</li>
  
<li>
เวเคน มคฺคํ คนฺตฺวาน, ปญฺจคาวุตมตฺตกํ;  
  
พทลตฺถลคามสฺส, ปเทเสนาติ ทูรเกฯ  
</li>
  
<li>
คาม เมกมุปาคญฺจิ, ปิลิํ วตฺถูติ สญฺญิตํ;  
  
ชนานํ สนฺนิปาตาย, นิชานํ โส กตาวธิฯ  
</li>
  
<li>
นิชาคมนโต ปุพฺพํ, ปฎิมเคฺค นิสีทิตุํ;  
  
ปฎิลทฺธนิโยคานํ, เยจิ เทวาคเต ตทาฯ  
</li>
  
<li>
ตหิํ ฐิเต โส ปสฺสิตฺวา, เอตฺตกา กินฺนุ อาคตา;  
  
อิติ ปุจฺฉิ กุมาโรถ, เตปิ ตํ อิท มพฺรวุํฯ  
</li>
  
<li>
โลกปฺปวตฺติํ สกลํ, ชานเนฺตนาปิ สามินา;  
  
กิเมว มุจฺจเต มจฺจุ-ภยํ เกสํ น วิชฺชติฯ  
</li>
  
<li>
พาลาตานุคโต สามิ, ฐิโต วยสิ อีทิเส;  
  
อชฺชาปิ หิ มุเข ตุยฺหํ, ขีรคโนฺธ ปวายติฯ  
</li>
  
<li>
น เหวตฺถิ วิสุํ วิตฺต-ชาตํ สงฺคหิตํ ตว;  
  
ตทญฺญา โจปกรณ-สามคฺคี เนว วิชฺชเตฯ  
</li>
  
<li>
จิรมฺปริจิตเตฺตหิ, ทฬฺหํ สารุฬภตฺติหิ;  
  
วินา’เมฺหหิ วิสุํ เกวา’นุคนฺตาโร ชนาตุวํฯ  
</li>
  
<li>
กิญฺจาคตานมมฺหากํ, ปิตา ตุยฺหํ นริสฺสโร;  
  
กาเรสฺสติ อิทํ นาม, สพฺพถา เนว ญายเตฯ  
</li>
  
<li>
อมฺหากมนฺตรามเคฺค  
, สโงฺข นาม จมูปติ;  
  
มหพฺพโล มหาวีโร, รชฺชสีมํ ตมาวสํฯ  
</li>
  
<li>
ปจฺจตฺถิเต ฐเปตฺวเญฺญ, เอเต กติปยา มยํ;  
  
อญฺญมญฺญมฺหิ นิยต-มาสงฺกี หทยา ภุสํฯ  
</li>
  
<li>
อรุณุคฺคมเวลา จ, สมาสนฺนตราธุนา;  
  
อิติ ภีติํ ปกาเสสุํ, ปเจฺจกํ หทยสฺสิตํฯ  
</li>
  
<li>
นิสมฺม เตสํ วจนํ, วิธาย มธุรํ สิตํ;  
  
วีตสโงฺก กุมาโร โส, มุขาเน’สํ วิโลกิยฯ  
</li>
  
<li>
จริตฺวาปิ มยา สทฺธิ-เมเต’โห กาลเมตฺตกํ;  
  
น ชานิํสุ มมํ สเพฺพ, เยสญฺหิ ภยมิทิสํฯ  
</li>
  
<li>
อิติ วตฺวา ภยํ เตสํ, วิโนเทตุ มุปฎฺฐิตํ;  
  
สีหนาทํ ตทา’กาสิ, มหนฺตํ สีหวิกฺกโมฯ  
</li>
  
<li>
ติฎฺฐนฺตุ มานุสา สเพฺพ, มยิ หตฺถคตายุเธ;  
  
สโกฺก เทวานมิโนฺทปิ, กุปิโต กิํ กริสฺสติฯ  
</li>
  
<li>
พาโลติ มํ จินฺตย ตํ, ชาตา โว กุมตีทิสี;  
  
ปริกฺขียติ เต ชาณา, นวโย’ติ น กิํ สุตํฯ  
</li>
  
<li>
อเชฺชว กาตุเมเกน, กมฺมุนา จินฺติเตน เม;  
  
สเทสปร เทสฎฺฐา, ภยภตฺตี ยถามยิฯ  
</li>
  
<li>
กริสฺสนฺติ ยถา เวทํ, ภยํ ตุเมฺห ชหิสฺสถ;  
  
ตถา รตฺติยเมตาย, วิภาตามย ขเณน เมฯ  
</li>
  
<li>
อุนฺนเต ทสฺสยิสฺสามิ, พุทฺธิ สาหสวิกฺกเม;  
  
อนุธาวติ มํ ตาต, เสเตหิ ยทิ โว ภยํฯ  
</li>
  
<li>
ปุรโต โหถ ตุเมฺหหิ, วตฺวา เต คหิตายุโธ;  
  
สาหเสกรโส วีโร, ตมฺหา นิกฺขมฺม คามโตฯ  
</li>
  
<li>
อุทยา’จลสีสฎฺฐํ, เชตุมาทิจฺจมณฺฑลํ;  
  
อปรํ รวิ พิมฺพํว, ปจฺฉิมา สา มุโขทิตํฯ  
</li>
  
<li>
เตชสา ปสรเนฺตน, ชนานํ ปวิกาสยํ;  
  
เนตมฺพุชวนํ ปาโต, พทลตฺถลิมาคมิฯ  
</li>
  
<li>
ชฆสงฺขสฺสเรนา’ถ, เสนา นาโถ ปพุชฺฌิย;  
  
สญฺชาตสมฺภโม ญตฺวา, ราชปุตฺตมุปาคตํฯ  
</li>
  
<li>
สทฺธิํ  
พเลน มหตา, ปธุคฺคมฺมกตาทโร;  
  
ปณามมุจิตํ กตฺตุ-มานโต วสุธาตเลฯ  
</li>
  
<li>
อมฺหากเมสชีวโนฺต, กิํ นามตฺถํ กริสฺสติ;  
  
มาเรตโพฺพ’ธุเนเวติ, ปสฺสเนฺต สมุขํ ภเฎฯ  
</li>
  
<li>
เนวา’ทิฎฺฐาปราธสฺส, มรณํ ปุริโส จิตํ;  
  
วโธ วิโรเธ สกฺกา’ติ, อิงฺคิเตน นิวาริยฯ  
</li>
  
<li>
เสนาปติสฺส โส หตฺถํ, คเหตฺวา สีหสนฺนิโภ;  
  
ภาสโนฺต มธุรํ วาจํ, ตเสฺสวา’คญฺฉิ มนฺทิรํฯ  
</li>
  
<li>
อถสฺส คมนํ รโญฺญ, ภวิตพฺพมชานตา;  
  
สรูปํ ยาว ชานามิ, ตาวเสฺสเต สหาคตาฯ  
</li>
  
<li>
ยถา น สหิตา โหนฺติ, ฐเปตพฺพา วิสุํ วิสุํ;  
  
กุมาโร’ว มมาคาเร, วสตู’ติ วิจินฺติยฯ  
</li>
  
<li>
ตถา เสนาปติ กตฺวา, วเญฺจตุํ ตํ มหามติํ;  
  
ทเสฺสตฺวา’ติ ถีสกฺการํ, รโญฺญ ทูเต ส เปสยิฯ  
</li>
  
<li>
กุมาโร’ถ วิทิตฺวาน, เตน ตํ วญฺจนํ กตํ;  
  
กตฺตพฺพเมตฺถา’กตฺวาห, มุทาสีโน ภเว ยทิฯ  
</li>
  
<li>
อิจฺฉิตตฺถสฺส นิปฺผตฺติ, น เม ชาตุ ภวิสฺสติ;  
  
อยํ ตาวา’ธุนาวสฺสํ, มาเรตโพฺพติ จินฺติยฯ  
</li>
  
<li>
สหาคตํ ปโยเชตฺวา, ฆาตาปยิ จมูปติํ;  
  
หโต เสนาธินาโถ’ติ, มหนฺตํ ขุภิตํ อหุฯ  
</li>
  
<li>
เสนานาถภโฎ เอโก, สุตฺวา เสนาปติํ หตํ;  
  
มารณํ สามิโน มยฺหํ, กิํ นิมิตฺตมีติ พฺรวิฯ  
</li>
  
<li>
เนตฺติํ สปาณี สหสา, กุมารํ ฐิตเมกกํ;  
  
อภิทาวิ สสามิสฺส, ปริจตฺตตฺตชีวิโนฯ  
</li>
  
<li>
กุมารสฺส มุขํ ทิสฺวา, เวธมาโน ภเยน โส;  
  
ปุเร ฐาตุมสโกฺกโนฺต, ปาทมูเล ตโต สยิฯ  
</li>
  
<li>
คณฺหเถ’ตนฺติวจนา-กุมารสฺส  
ปุเรตรํ;  
  
ตเสฺสเวโก สหจโร, ภฎเมตํ วิฆาตยิฯ  
</li>
  
<li>
นิโยคํ เม วินา เตน, กตํ กมฺมํ น ยุชฺชติ;  
  
อิติ ทณฺฑนเมตสฺส, การาเปสิยโถจิตํฯ  
</li>
  
<li>
อถ ตํ กาลยมฺภูต-สโงฺขภมติภิํสนํ;  
  
ภมุกฺขิปนมเตฺตน, ราชปุโตฺต สมํ นยิฯ  
</li>
  
<li>
วีโร ยโสธรธโน ธิติมา กุมาโร;  
  
วีโรปการจตุโร วรกิตฺติสาโร;  
  
เสนินฺทสญฺจิตมนปฺปธนํ ภฎานํ;  
  
สพฺพํ วิสชฺชยิ ยถารุจิยา คเหตุํฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
เสนาปติวโธ นาม  
</li>
  
ติสฎฺฐิโม ปริเจฺฉโทฯ  
</li>

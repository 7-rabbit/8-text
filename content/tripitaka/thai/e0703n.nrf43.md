<h1>จตุจตฺตาลีสติม ปริเจฺฉท</h1>
<h3>ติราชโก</h3>
<ol>
<li>
อจฺจเย  
หตฺถทาฐสฺส, อคฺคโพธิกุมารโก;  
  
กณิโฎฺฐ ราชิโน อาสิ, สิริสงฺฆาทิ โพธิโกฯ  
</li>
  
<li>
ธมฺมราชา อยํ อาสิ, สมฺมา ทสฺสนสญฺญุโต;  
  
ตสฺมา โส ปุญฺญกมฺมานิ, อปฺปเมยฺยานิ วตฺตยิฯ  
</li>
  
<li>
นิกายตฺตยวาสีนํ, ภตฺตคฺคมวโลกยิ;  
  
มหาปาฬิญฺจ วเฑฺฒสิ, มาฆาตเญฺจว การยิฯ  
</li>
  
<li>
ฐานนฺตรญฺจ ทาเปสิ, ยถารห มนาลโย;  
  
สิปโคตฺตาทิโยเคฺคหิ, สํคเตหิ จ สํคหิฯ  
</li>
  
<li>
ยตฺถ กตฺถจิ ทิสฺวาปิ, ภิกฺขโว โส มหามติ;  
  
สกฺกตฺวา โสภณาเปสิ, ปริตฺตํ สาสโนคธํฯ  
</li>
  
<li>
เถรํ โส อุปสงฺกมฺม, นาคสาลนิวาสินํ;  
  
ทาฐาสิวํ มหาปญฺญํ, สีลวนฺตํ พหุสฺสุตํฯ  
</li>
  
<li>
สกฺกจฺจ นํ ตโต สุตฺวา, สมฺมาสมฺพุทฺธสาสนํ;  
  
ธเมฺม’ตีวปสีทิตฺวา, สพฺพสนฺติกโร อิติฯ  
</li>
  
<li>
สุตฺวา เถริยวาทานํ, ปุพฺพญาตีนมตฺตโน;  
  
ปาปานํ ทุฎฺฐจิตฺตานํ, อปกาเร กเต พหูฯ  
</li>
  
<li>
วิหาเร ปริเวเณ จ, ชิเณฺณปากติเก อกา;  
  
โภคคาเม จ ทาเปสิ, ตตฺถ ตตฺถ พหูทเยฯ  
</li>
  
<li>
วิจฺฉินฺนปจฺจเย จากา, ตทาตฺยํ กุริเต วิย;  
  
ทาสเกปิ จ สงฺฆสฺส, ยถาฐาเน ฐปาปยิฯ  
</li>
  
<li>
ปธานฆร เมตสฺส, เถรสฺส’กาสนามกํ;  
  
ปฎิคฺคเหตฺวา ตํ โส’ปิ, สงฺฆสฺสา’ทา มหามตีฯ  
</li>
  
<li>
โภคคาเม จ ตสฺสา’ทา, ภรตฺตาลํ กิหิมฺพิลํ;  
  
กตกญฺจ ตุลาธารํ, อนฺธการกเมว จฯ  
</li>
  
<li>
อนฺธการํ  
อนฺตุเรฬิํ, พาลวํ ทฺวารนายกํ;  
  
มหานิกฑฺฒิกเญฺจว, เปฬหาฬํ ตถาปรํฯ  
</li>
  
<li>
เอเต อเญฺญ จ โส ทตฺวา, โภคคาเม นริสฺสโร;  
  
ทาสิ อารามิเก เจว, อตฺตโน กิร ญาตเกฯ  
</li>
  
<li>
ตถา ทฺวินฺนํ นิกายานํ, วิหาเร มนฺทปจฺจเย;  
  
ทิสฺวาปิ จ สุตฺวา วา, โภคคาเม พหู อทาฯ  
</li>
  
<li>
พหุนา กินฺตุวุเตฺตน, นิกาเยสุ’ปิ ตีสุ’ปิ;  
  
อทา คามสหสฺสํ โส, พหุปฺปาทํ นิรากุลํฯ  
</li>
  
<li>
อนุสฺสรโนฺต โส ติณฺณํ, รตนานํ คุเณ วเร;  
  
เอกาวลิํ คเหตฺวาน, อกฺขมาล มกา กิรฯ  
</li>
  
<li>
เอวํ สพฺพปฺปโยเคหิ, โส’หุ ธมฺมปรายโน;  
  
สเพฺพ ตมนุสิกฺขนฺตา, เหสุ ธมฺมกรา นราฯ  
</li>
  
<li>
ทมิโฬ โปตฺถกุฎฺฐโว, ตสฺส กมฺมกโร ตทา;  
  
มาฎมฺพิยวฺหํ กาเรสิ, ปธานฆรมพฺภุตํฯ  
</li>
  
<li>
พูกกเลฺล อมฺพวาปิํ, ตนฺตวายิกจาฎิกํ;  
  
คามํ นิฎฺฐิลเวฎฺฐิญฺจ, ตสฺสา’ทา โส สทา สกํฯ  
</li>
  
<li>
กปฺปูรปริเวเณ จ, กุรุนฺทปิลฺลเก ตถา;  
  
มหาราชฆเร เจว, ปาสาเท โส’ว การยิฯ  
</li>
  
<li>
อญฺญตฺถา’ทา ตโย คาเม, วิหาเรสุ มหาธโน;  
  
โปตฺถสาตวฺหโย ปโญฺญ, วิหาเร เชตนามเกฯ  
</li>
  
<li>
เสนาปติราชนามํ, ปริเวณํ สมาปยิ;  
  
มหากโนฺท จ ทมิโฬ, ปริเวณํ สนามกํฯ  
</li>
  
<li>
จุลฺลปนฺถํ ตถา เอโก, เสหาลอุปราชกํ;  
  
อุปราชา สกาเรสิ, สงฺฆติโสฺส’ปิ ราชิโนฯ  
</li>
  
<li>
อเญฺญสุ พหโว อาสุํ, วิหาเร เอวมาทิเก;  
  
ภสฺส รโญฺญ’ นุวตฺตนฺตา, เอวํ ธมฺมีหิ ปาณิโนฯ  
</li>
  
<li>
ปาปํ วาปิ หิ ปุญฺญํ วา,  
  
ปธาโนยํ กโรติ โย;  
  
โล โก ตํ ตํ กโรเตว,  
  
ตํ วิชเนยฺย ปณฺฑิโตฯ  
</li>
  
<li>
เชฎฺฐนามา  
มหาปุญฺญา, มเหสี ตสฺส ราชิโน;  
  
เชฎฺฐารามญฺจ กาเรสิ, ภิกฺขูนีนมุปสฺสยํฯ  
</li>
  
<li>
ตสฺสา’ทาสิ จ เทฺว คาเม, ปกฺกปาสาณภูมิยํ;  
  
ตมฺพุทฺธํ เภลคามญฺจ, อารามิกสตํ ตถาฯ  
</li>
  
<li>
อกา มลยราชาปิ, ธาตุเคหํ มหารหํ;  
  
มณฺฑลคิริวิหารมฺหิ, เจติยสฺส มหาธโนฯ  
</li>
  
<li>
โลหปาสาทเก โส’ว, ฉาเทสิ มชฺฌกูฎกํ;  
  
โพธิติสฺสวิหารญฺจ, โพธิตโสฺส มหายโสฯ  
</li>
  
<li>
ทีเป มณฺฑลิกา สเพฺพ, ตตฺถ ตตฺถ ยถาพลํ;  
  
วิหาเร ปริเวเณ จ, การยิํสุ อนปฺปเกฯ  
</li>
  
<li>
ตสฺส กาโล นรินฺทสฺส, ปุญฺญกมฺมมโย อิธ;  
  
อติวิตฺถารภีเตน, สพฺพโส น วิจาริตํฯ  
</li>
  
<li>
ปุพฺพโกปิ กถามโคฺค, อากุโล วิย ภาติเม;  
  
ยถาปธานํ กถิตํ, เหตูนํ อุปลกฺขณํฯ  
</li>
  
<li>
อถาปเรน กาเลน, ปุลตฺถินครํ คโต;  
  
วาสํ ตเตฺถว กเปฺปสิ, กโรโนฺต ปุญฺญสญฺจยํฯ  
</li>
  
<li>
อเตกิจฺฉิยโรเคน, สมฺผุโฎฺฐ กาลมตฺตโน;  
  
มรณสฺส วิทิตฺวาน, ตมาหุยมหาชนํฯ  
</li>
  
<li>
โอวทิตฺวาน ธเมฺมน, มรณํ โส อุปาคมิ;  
  
มหาชโน มเต ตสฺมิํ, พาฬฺหโสโก ปโรทิยฯ  
</li>
  
<li>
กตฺวา อาฬหเณ ตสฺส, กิจฺจํ สพฺพมเสสโต;  
  
ตสฺสาฬาหณภสฺมมฺปิ, กตฺวา เภสชฺชมตฺตโนฯ  
</li>
  
<li>
ราชา ภณฺฑญฺจ ตํ สพฺพํ, สพฺพญฺจ พลวาหนํ;  
  
สมฺมา อาทาย โคเปตฺวา, นครํ สมุปาคมิฯ  
</li>
  
<li>
เอวํ โสฬสเม วเสฺส, ราชา อาสิ ทิวงฺคโต;  
  
โปตฺถกุโฎฺฐปิ ทมิโฬ, รชฺชํ ตสฺส วิจารยิฯ  
</li>
  
<li>
อุปราชํ คเหตฺวาน, ทาฐาสิวํ ขิปาปยิ;  
  
จารโก วิหิตุํ สมฺมา, รกฺขาวรณ มาทิสิฯ  
</li>
  
<li>
วินา รญฺญา น สกฺกาติ, เมทินิํ ปริภุญฺชิตุํ;  
  
อาเนตฺวา ทตฺตนามกํ, ธนปิฎฺฐปฺปธานกํฯ  
</li>
  
<li>
อุปฺปนฺนํ  
ราชวํสมฺหิ, รเชฺช ตํ อภิสิญฺจิย;  
  
ตสฺส นามํ ฐเปตฺวาน, สยํ สพฺพํ วิจารยิฯ  
</li>
  
<li>
ทโนฺต โส ธนปิฎฺฐิมฺหิ, วิหารํ สกนามกํ;  
  
การยิตฺวาน ปุญฺญานิ, อญฺญานิปิ สมาจินิฯ  
</li>
  
<li>
สมกนฺตุ โส ฐตฺวา’ว, วสฺสทฺวย มหุมโต;  
  
โปตฺถกุโฎฺฐ มเตตสฺมิํ, ปุน อญฺญมฺปิ มาณวํฯ  
</li>
  
<li>
หตฺถทาฐํ สมาหูย, อุณฺหา นครสมฺภวํ;  
  
ตมฺปิ รเชฺช ภิสิญฺจิตฺวา, ยถา ปุพฺพํ สยํวสีฯ  
</li>
  
<li>
กาฬทีฆาวิกํ กตฺวา, ปธานฆรกํ ตถา;  
  
ปุญฺญมญฺญํ ฉมาเสหิ, โส’ปิ มจฺจุวสํ คโตฯ  
</li>
  
<li>
เอวํ วิทิตฺวา พหุปทฺทวานิ;  
  
ธนานิ ธญฺญานิ จ วาหนานิ;  
  
วิหาย รเชฺชสุ รติํสปฺปญฺญา;  
  
มนุญฺญปุญฺญาภิรตา ภเวยฺยุํฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
ติราชโก นาม  
</li>
  
จตุจตฺตาลีสติโม ปริเจฺฉโทฯ  
</li>

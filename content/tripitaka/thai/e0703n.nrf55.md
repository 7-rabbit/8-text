<h1>ฉปญฺญาสติม ปริเจฺฉท</h1>

</li>

</li>
  
<li>
เสนินฺทํ กาชรคฺคาม-สมีปํ สมุปาคตํ;

</li>
  
<li>
เสนิโนฺท กาชรคฺคามํ, วิลุมฺปิตฺวาน เวคสา;

</li>
  
<li>
ตโต มหาทิปาโท’ปิ, อาคมฺม มลยา ลหุํ;

</li>
  
<li>
ราชา รามญฺญวิสเย, ราชิโน สนฺติกํ ชเน;

</li>
  
<li>
วิจิตฺตวตฺถกปฺปูร-จนฺทนาทีหิ วตฺถูหิ;  
  
ปริปุณฺณา จ นาวาโย, เนกาติเตฺถ สโมสฎาฯ  
</li>
  
<li>
ธนชา เตหิ เนเกหิ, กโรโนฺต ภฎสงฺคหํ;  
  
ตโต ตมฺพลคามมฺหิ, นิวสิตฺถ มหพฺพโลฯ  
</li>
  
<li>
อญฺญมญฺญวิรุทฺธตฺตา, ราชารฎฺฐาธิวาสิโน;  
  
อุปคฺคมฺม ชนา สเพฺพ, กรํ โน เทนฺติ สพฺพโสฯ  
</li>
  
<li>
วิปกฺขาโจฬราชสฺส, ภินฺทนฺตา’ณํ มทุทฺธตา;  
  
อายุตฺตเก วิหิํเสโนฺต, ยถากามํ จรนฺติ จฯ  
</li>
  
<li>
ตํ  
สุตฺวา โจฬภูปาโล, สมฺปโกปวสีกโต;  
  
เสนํ มหนฺติํ ทตฺวาน, เปเสสิ สจิวํ นิชํฯ  
</li>
  
<li>
มหาติเตฺถ’วหิโณฺณ โส, ตตฺถ ตตฺถ พหูชเน;  
  
ฆาเตโนฺต สวเส กตฺวา, ราชารฎฺฐาธิวาสิโนฯ  
</li>
  
<li>
อนุกฺกเมน คนฺตฺวาน, ขราโณ โรหณํ ตทา;  
  
อโชฺฌตฺถริ สเสโน ส, ภิโนฺน เวโลวสาคโรฯ  
</li>
  
<li>
รโญฺญ ปจฺจตฺถิกา หุตฺวา, รวิเทวจลวฺหยา;  
  
อุโภ ทมิฬเสนินฺท-วสํ ยาตา มหพฺพลาฯ  
</li>
  
<li>
มหาปกฺขพโลเปเต, เต ปสฺสิย จมูปติ;  
  
โรหณํ สกลํ ขิปฺปํ, มญฺญิตฺถ สกหตฺถคํฯ  
</li>
  
<li>
เอกาทสมฺหิ โส วเสฺส, ราชา โจฬาภิภูติยา;  
  
ปลุฎฺฐปพฺพเต ทุคฺคํ, ปวิธาย ตหิํ วสีฯ  
</li>
  
<li>
โจฬเสนา ตทาเสลํ, สมนฺตา สมฺปริกฺขิปิ;  
  
ตโตฺถ’ภเยสํ เสนาน-มหู ภิํสนกํ รณํฯ  
</li>
  
<li>
รโญฺญ ภฎา วินาเสตฺวา, สพฺพํ ตํ ทมิฬํ พลํ;  
  
ปลายนฺตํ มหาโจฬ-สามนฺตญฺจานุพนฺธิยฯ  
</li>
  
<li>
คเหตฺวาน สิรํ ตสฺส, คามสฺมิํ ตมฺพวิฎฺฐิเก  
  
สทฺธิํ วาหนยาเนหิ, สารภูตธเนหิ จฯ  
</li>
  
<li>
ตหิํ ตํ สกลํ ภณฺฑํ, เนตฺวา รโญฺญ ปทสฺสิย;  
  
ปุฬตฺถินครํ คนฺตุํ, กาโล อิติ สมพฺรวุํฯ  
</li>
  
<li>
มหีปาโลปิ ตํ สุตฺวา, สจิวานํ วโจ ตทา;  
  
มหตา พลกาเยน, ปุลตฺถินครํ คโตฯ  
</li>
  
<li>
ปวตฺติเมตํ สกลํ, สุตฺวา โจฬนริสฺสโร;  
  
ติพฺพโกปวสํยาโต, ภูปาลคหณตฺถิโกฯ  
</li>
  
<li>
สามํ ขิปฺปํ สมาคมฺม, วีโร สาครปฎฺฎนํ;  
  
ภิโยฺยปิ มหติํ เสนํ, ลํกาทีปมเปสยีฯ  
</li>
  
<li>
ตํ วิชานิย ภูมิโนฺท, เสนินฺทํ เปสยี สกํ;  
  
พลํ มหนฺตํ ทตฺวาน, โจฬเสนาย ยุชฺฌิตุํฯ  
</li>
  
<li>
คจฺฉมาโน จมูนาโถ, โส’นุราธปุรนฺติเก;  
  
สทฺธิํ ทมิฬเสนาย, วเตฺตสิ ตุมุลํ รณํฯ  
</li>
  
<li>
ปติํสุ  
ตสฺมิํ สงฺคาเม, ภูปาลสฺส นรา พหู;  
  
ภิโยฺยปิ ทมิฬายตฺตา, ชาตา ตํ รฎฺฐวาสิโนฯ  
</li>
  
<li>
วิหาย ธรณีปาโล, ปุลตฺถินครํ ตทา;  
  
วิลฺลิกาพาณกํ รฎฺฐํ, สมฺปาปุณิย เวคสาฯ  
</li>
  
<li>
นิหจฺจามจฺจยุคลํ, ตํ รฎฺฐาธิฎฺฐิตํ ตทา;  
  
ตหิํ วาสมกเปฺปสิ, สภเฎ สนฺนิปาตยํฯ  
</li>
  
<li>
อตฺตานมนุพนฺธนฺตํ, สุตฺวา โจฬจมูปติํ;

</li>
  
<li>
อุปจฺจกาย เสลสฺส, ตตฺถ ทุคฺคํ นิเวสิย;

</li>
  
<li>
เกสธาตุกนาถสฺส, มาริตสฺส ปุรารเณ;

</li>
  
<li>
มารณํ สกภาตุสฺส, สรโนฺต รุฎฺฐมานโส;  
  
สกลํ ปริวเตฺตสิ, คุตฺตหาลกมณฺฑลํฯ  
</li>
  
<li>
อโถ ลํกิสฺสโร ตตฺถ, ขิปฺปํ คนฺตฺวา มหพฺพโล;  
  
ฐาเน มจฺจุตฺถเล นาม, ขนฺธาวารํ นิพนฺธิยฯ  
</li>
  
<li>
ขทิรงฺคณิทุคฺคมฺหา, ปลาเปตฺวาน ตํ รเณ;

</li>
  
<li>
วิหาย ปุตฺตทาราทิ-สพฺพํ เนกธนํ พลํ;

</li>
  
<li>

</li>
  
<li>

<li>

<li>

</li>
  
<li>

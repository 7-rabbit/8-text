<h1>๑๓. สุเมธพุทฺธวํโส</h1>
<ol>
<li>
ปทุมุตฺตรสฺส  
  
  
อปเรน, สุเมโธ นาม นายโก;  
  
ทุราสโท อุคฺคเตโช, สพฺพโลกุตฺตโม มุนิฯ  
</li>
  
<li>
ปสนฺนเนโตฺต สุมุโข, พฺรหา อุชุ ปตาปวา;  
  
หิเตสี สพฺพสตฺตานํ, พหู โมเจสิ พนฺธนาฯ  
</li>
  
<li>
ยทา พุโทฺธ ปาปุณิตฺวา, เกวลํ โพธิมุตฺตมํ;  
  
สุทสฺสนมฺหิ นคเร, ธมฺมจกฺกํ ปวตฺตยิฯ  
</li>
  
<li>
ตสฺสาปิ อภิสมยา ตีณิ, อเหสุํ ธมฺมเทสเน;  
  
โกฎิสตสหสฺสานํ, ปฐมาภิสมโย อหุฯ  
</li>
  
<li>
ปุนาปรํ กุมฺภกณฺณํ, ยกฺขํ โส ทมยี ชิโน;  
  
นวุติโกฎิสหสฺสานํ, ทุติยาภิสมโย อหุฯ  
</li>
  
<li>
ปุนาปรํ อมิตยโส, จตุสจฺจํ ปกาสยิ;  
  
อสีติโกฎิสหสฺสานํ, ตติยาภิสมโย อหุฯ  
</li>
  
<li>
สนฺนิปาตา ตโย อาสุํ, สุเมธสฺส มเหสิโน;  
  
ขีณาสวานํ วิมลานํ, สนฺตจิตฺตาน ตาทินํฯ  
</li>
  
<li>
สุทสฺสนํ  
นาม นครํ, อุปคญฺฉิ ชิโน ยทา;  
  
ตทา ขีณาสวา ภิกฺขู, สมิํสุ สตโกฎิโยฯ  
</li>
  
<li>
ปุนาปรํ เทวกูเฎ, ภิกฺขูนํ กถินตฺถเต;  
  
ตทา นวุติโกฎีนํ, ทุติโย อาสิ สมาคโมฯ  
</li>
  
<li>
ปุนาปรํ  
ทสพโล, ยทา จรติ จาริกํ;  
  
ตทา อสีติโกฎีนํ, ตติโย อาสิ สมาคโมฯ  
</li>
  
<li>
อหํ เตน สมเยน, อุตฺตโร นาม มาณโว;  
  
อสีติโกฎิโย มยฺหํ, ฆเร สนฺนิจิตํ ธนํฯ  
</li>
  
<li>
เกวลํ สพฺพํ ทตฺวาน, สสเงฺฆ โลกนายเก;  
  
สรณํ ตสฺสุปคญฺฉิํ, ปพฺพชฺชญฺจาภิโรจยิํฯ  
</li>
  
<li>
โสปิ มํ พุโทฺธ พฺยากาสิ, กโรโนฺต อนุโมทนํ;  
  
‘‘ติํสกปฺปสหสฺสมฺหิ, อยํ พุโทฺธ ภวิสฺสติฯ  
</li>
  
<li>
‘‘ปธานํ  
ปทหิตฺวาน…เป.…  
เหสฺสาม สมฺมุขา อิมํ’’ฯ  
  
<li>
ตสฺสาปิ วจนํ สุตฺวา, ภิโยฺย จิตฺตํ ปสาทยิํ;  
  
อุตฺตริํ วตมธิฎฺฐาสิํ, ทสปารมิปูริยาฯ  
</li>
  
<li>
สุตฺตนฺตํ วินยญฺจาปิ, นวงฺคํ สตฺถุสาสนํ;  
  
สพฺพํ ปริยาปุณิตฺวาน, โสภยิํ ชินสาสนํฯ  
</li>
  
<li>
ตตฺถปฺปมโตฺต  
  
วิหรโนฺต, นิสชฺชฎฺฐานจงฺกเม;  
  
อภิญฺญาสุ ปารมิํ คนฺตฺวา, พฺรหฺมโลกมคญฺฉหํฯ  
</li>
  
<li>
สุทสฺสนํ นาม นครํ, สุทโตฺต นาม ขตฺติโย;  
  
สุทตฺตา นาม ชนิกา, สุเมธสฺส มเหสิโนฯ  
</li>
  
<li>
นววสฺสสหสฺสานิ, อคารํ อชฺฌ โส วสิ;  
  
สุจนฺทกญฺจนสิริวฑฺฒา, ตโย ปาสาทมุตฺตมาฯ  
</li>
  
<li>
ติโสฬสสหสฺสานิ, นาริโย สมลงฺกตา;  
  
สุมนา นาม สา นารี, ปุนพฺพสุ นาม อตฺรโชฯ  
</li>
  
<li>
นิมิเตฺต จตุโร ทิสฺวา, หตฺถิยาเนน นิกฺขมิ;  
  
อนูนกํ อฑฺฒมาสํ, ปธานํ ปทหี ชิโนฯ  
</li>
  
<li>
พฺรหฺมุนา ยาจิโต สโนฺต, สุเมโธ โลกนายโก;  
  
วตฺติ จกฺกํ มหาวีโร, สุทสฺสนุยฺยานมุตฺตเมฯ  
</li>
  
<li>
สรโณ  
สพฺพกาโม จ, อเหสุํ อคฺคสาวกา;  
  
สาคโร นามุปฎฺฐาโก, สุเมธสฺส มเหสิโนฯ  
</li>
  
<li>
รามา เจว สุรามา จ, อเหสุํ อคฺคสาวิกา;  
  
โพธิ ตสฺส ภควโต, มหานีโปติ วุจฺจติฯ  
</li>
  
<li>
อุรุเวลา ยสวา จ, อเหสุํ อคฺคุปฎฺฐกา;  
  
ยโสธรา สิริมา จ  
ยสา นาม สิริมา จ (สฺยา. กํ.)  
, อเหสุํ อคฺคุปฎฺฐิกาฯ  
</li>
  
<li>
อฎฺฐาสีติรตนานิ, อจฺจุคฺคโต มหามุนิ;  
  
โอภาเสติ ทิสา สพฺพา, จโนฺท ตารคเณ ยถาฯ  
</li>
  
<li>
จกฺกวตฺติมณี  
นาม, ยถา ตปติ โยชนํ;  
  
ตเถว ตสฺส รตนํ, สมนฺตา ผรติ โยชนํฯ  
</li>
  
<li>
นวุติวสฺสสหสฺสานิ  
, อายุ วิชฺชติ ตาวเท;  
  
ตาวตา ติฎฺฐมาโน โส, ตาเรสิ ชนตํ พหุํฯ  
</li>
  
<li>
เตวิชฺชฉฬภิเญฺญหิ, พลปฺปเตฺตหิ ตาทิหิ;  
  
สมากุลมิทํ อาสิ, อรหเนฺตหิ สาธุหิฯ  
</li>
  
<li>
เตปิ สเพฺพ อมิตยสา, วิปฺปมุตฺตา นิรูปธี;  
  
ญาณาโลกํ ทสฺสยิตฺวา, นิพฺพุตา เต มหายสาฯ  
</li>
  
<li>
สุเมโธ ชินวโร พุโทฺธ, เมธารามมฺหิ นิพฺพุโต;  
  
ธาตุวิตฺถาริกํ อาสิ, เตสุ เตสุ ปเทสโตติฯ  
</li>
  
สุเมธสฺส ภควโต วํโส เอกาทสโมฯ  
</li>

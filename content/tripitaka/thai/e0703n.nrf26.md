<h1>สตฺตวีสติม ปริเจฺฉท</h1>
<h3>โลหปาสาทมโห</h3>
<ol>
<li>
ตโต ราชา วิจิเนฺตสิ, วิสฺสุตํ สุสฺสุตํ สุตํ;  
  
มหาปโญฺญ สทา ปุโญฺญ, ปญฺญาย กตนิจฺฉโยฯ  
</li>
  
<li>
‘‘ทีปปฺปสาทโก เถโร, ราชิโน อยฺยกสฺส เม;  
  
เอวํ กิราหนตฺตา เต, ทุฎฺฐคามณิภูปติฯ  
</li>
  
<li>
มหาปุโญฺญ มหาถูปํ, โสณฺณมาลิํ มโนรมํ;  
  
วีสํหตฺถสตํ อุจฺจํ, กาเรสฺสติ อนาคเตฯ  
</li>
  
<li>
ปุโน อุโปสถาคารํ, นานารตนมณฺฑิตํ;  
  
นวภูมํ กริตฺวาน, โลหปาสาท เมว จฯ  
</li>
  
<li>
อิติ จินฺติย ภูมิโนฺท, ลิขิเตฺววํ ฐปาปิตํ;  
  
เปกฺขาเปโนฺต ราชเคเห, ฐิตํ เอว กรณฺฑเกฯ  
</li>
  
<li>
โสวณฺณปฎฺฎํ ลทฺธาน, เลขํ ตตฺถ อวาจยิ;  
  
‘‘จตฺตาลีสสตํ วสฺสํ, อติกฺกมฺม อนาคเตฯ  
</li>
  
<li>
กากวณฺณสุโต  
ทุฎฺฐ-คามณิมนุชาธิโป;  
  
อิทญฺจิทญฺจ เอวญฺจ, กาเรสฺสตี’’ติ วาจิตํฯ  
</li>
  
<li>
สุตฺวา หโฎฺฐ อุทาเนตฺวา, อโปฺปเฎฺฐสิ มหีปติ;  
  
ตโต ปาโต’ว คนฺตฺวาน, มหาเมฆวนํ สุภํฯ  
</li>
  
<li>
สนฺนิปาตํ การยิตฺวา, ภิกฺขุสงฺฆสฺส อพฺรวิ;  
  
‘‘วิมานตุลฺยํ ปาสาทํ, การยิสฺสามิ โว อหํฯ  
</li>
  
<li>
ทิพฺพํ วิมานํ เปเสตฺวา, ตทา เลขํ อทาถ เม’’;  
  
ภิกฺขุสโงฺฆ วิสเชฺชสิ, อฎฺฐ ขีณาสเว ตหิํฯ  
</li>
  
<li>
กสฺสปมุนิโน กาเล, อโสโก นาม พฺราหฺมโณ;  
  
อฎฺฐ สลากภตฺตานิ, สงฺฆสฺส ปริณามิยฯ  
</li>
  
<li>
ภรณิํ นาม ทาสิํ โส, ‘‘นิจฺจํ เทหี’’ติ อพฺรวิ;  
  
ทตฺวา สา ตานิ สกฺกจฺจํ, ยาวชีวํ ตโต จุตาฯ  
</li>
  
<li>
อากาสฎฺฐวิมานมฺหิ, นิพฺพตฺติรุจิเร สุเภ;  
  
อจฺฉรานํ สหเสฺสน, สทา’สิ ปริวาริตาฯ  
</li>
  
<li>
ตสฺสา รตนปาสาโท, ทฺวาทสโยชนุคฺคโต;  
  
โยชนานํ ปริเกฺขโป, จตฺตาลีสญฺจ อฎฺฐ จฯ  
</li>
  
<li>
กูฎาคารสหเสฺสน, มณฺฑิโต นวภูมิโก;  
  
สหสฺสคพฺภสมฺปโนฺน, ราชมาโน จตุมฺมุโขฯ  
</li>
  
<li>
สหสฺสสงฺขสํวุตฺติ, สีหปญฺชร เนตฺตวา;  
  
สกิงฺกณิกชาลาย, สชฺชิโต เวทิกาย จฯ  
</li>
  
<li>
อมฺพลฎฺฐิกปาสาโท, ตสฺส มเชฺฌ ฐิโต อหุ;  
  
สนฺตมโต ทิสฺสมาโน, ปคฺคหิตธชากุโลฯ  
</li>
  
<li>
ตาวติํสญฺจ คจฺฉนฺตา, ทิสฺวา เถรํ ตเมว เต;  
  
หิงฺคุเลน ตทา เลขํ, เลขยิตฺวา ปเฎ ตโตฯ  
</li>
  
<li>
นิวตฺติตฺวาน อาคนฺตฺวา, ปฎํ สงฺฆสฺส ทสฺสยุํ;  
  
สโงฺฆ ปฎํ คเหตฺวา ตํ, ปาเหสิ ราชสนฺติกํฯ  
</li>
  
<li>
ตํ  
ทิสฺวา สุมโน ราชา, อาคมฺมาราม มุตฺตมํ;  
  
อาเลขตุลฺยํ กาเรสิ, โลหปาสาท มุตฺตมํฯ  
</li>
  
<li>
กมฺมารมฺภนกาเลว, จตุทฺวารมฺหิ โภควา;  
  
อฎฺฐสตสหสฺสานิ, หิรญฺญานิ ฐปาปยิฯ  
</li>
  
<li>
ปุฎสหสฺส วตฺถานิ, ทฺวาเร ทฺวาเร ฐปาปยิ;  
  
คุฬ เตลสกฺขรมธุ-ปุรา จาเนก จาฎิโยฯ  
</li>
  
<li>
‘‘อมูลกกมฺม เมตฺถ, น กาตพฺพ’’นฺติ ภาสิย;  
  
อคฺฆาเปตฺวา กตํ กมฺมํ, เตสํ มูลมทาปยิฯ  
</li>
  
<li>
หตฺถสตํ หตฺถสตํ, อาสิ เอเกก ปสฺสโต;  
  
อุจฺจโต ตตฺตโกเยว, ปาสาโท หิ จตุมฺมุโขฯ  
</li>
  
<li>
ตสฺมิํ ปาสาท เสฎฺฐสฺมิํ, อเหสุํ นวภูมิโย;  
  
เอเกกิสฺสา ภูมิยา จ, กูฎาคารสตานิ จฯ  
</li>
  
<li>
กูฎาคารานิ สพฺพานิ, สชฺฌุนา ขจิตา น’ยุํ;  
  
ปวาลเวทิกา เตสํ, นานารตน ภูสิตาฯ  
</li>
  
<li>
นานารตน จิตฺตานิ, ตาสํ ปทุมกานิ จ;  
  
สชฺฌกิํกิณิกาปนฺติ-ปริกฺขิตฺตาว ตา อหุฯ  
</li>
  
<li>
สหสฺสํ ตตฺถ ปาสาโท, คพฺภา อาสุํ สุสงฺขตา;  
  
นานารตน ขจิตา, สีหปญฺชรเนตฺตวาฯ  
</li>
  
<li>
นาริวาหนยานนฺตุ, สุตฺวา เวสฺสวณสฺส โส;  
  
ตทา การมกาเรสิ, มเชฺฌ รตนมณฺฑปํฯ  
</li>
  
<li>
สีหพฺยคฺฆาทิรูเปหิ, เทวตา รูปเกหิ จ;  
  
อหุ รตนมโยเห’ส, ถเมฺภหิ จ วิภูสิโตฯ  
</li>
  
<li>
มุตฺตาชาลปริเกฺขโป, มณฺฑปเนฺต สมนฺตโต;  
  
ปวาลเวทิกา เจตฺถ, ปุเพฺพ วุตฺตวิธา อหุฯ  
</li>
  
<li>
สตฺตรตน จิตฺตสฺส, เวมเชฺฌ มณฺฑปสฺส ตุ;  
  
รุจิโร ทนฺตปลฺลโงฺก, รโมฺม ผลิกสนฺถโรฯ  
</li>
  
<li>
ทนฺตมยาปสฺสเย’ตฺถ, สุวณฺณมย สูริโย;  
  
สชฺฌุมเย จนฺทิมา จ, ตารา จ มุตฺตกา มยาฯ  
</li>
  
<li>
นานารตน  
ปทุมานิ, ตตฺถ ตตฺถ ยถารหํ;  
  
ชาตกานิ จ ตเตฺถว, อาสุํ โสณฺณลตนฺตเรฯ  
</li>
  
<li>
มหคฺฆปจฺจตฺถรเณ, ปลฺลเงฺก’ติ มโนรเม;  
  
มโนหรา’สิฎฺฐปิตา, รุจิรํ ทนฺตพีชนีฯ  
</li>
  
<li>
ปวาลปาทุกา ตตฺถ, ผลิกมฺหิ ปติฎฺฐิตา;  
  
เสตจฺฉตฺตํ สชฺฌุทณฺฑํ, ปลฺลโงฺก’ปริโสภถฯ  
</li>
  
<li>
สตฺตรตน มยาเนตฺถ, อฎฺฐมงฺคลิกานิ จ;  
  
จตุปฺปทานํ ปนฺตี จ, มณิมุตฺตนฺตรา อหุํฯ  
</li>
  
<li>
รชตานญฺจ ฆณฺฎานํ, ปนฺตี ภตฺตนฺตลมฺพิตา;  
  
ปาสาทฉตฺตปลฺลงฺก-มณฺฑปา’สุํ อนคฺฆิกาฯ  
</li>
  
<li>
มหคฺฆ ปญฺญปาเปสิ, มญฺจปีฐํ ยถารหํ;  
  
ตเถว ภูมตฺถรณํ, กมฺพลญฺจ มหารหํฯ  
</li>
  
<li>
อาจาม กุมฺภิโสวณฺณา, อุลุโงฺก จ อหุ ตหิํ;  
  
ปาสาท ปริโภเคสุ, เสเสสุ จ กถา’ วกาฯ  
</li>
  
<li>
จารุปาการ ปริวาโร,  
  
โส จตุทฺวารโกฎฺฐโก;  
  
ปาสาโท’ลงฺกโต โสภิ,  
  
ตาวติํส สภา วิยฯ  
</li>
  
<li>
ตมฺพ โลหิฎฺฐกาเห’โส,  
  
ปาสาโท ฉาทิโต อหุ;  
  
โลห ปาสาท โวหาโร,  
  
เตน ตสฺส อชายถฯ  
</li>
  
<li>
นฎฺฐิเต โลหปาสาเท, โส สงฺฆํ สนฺนิปาตยิ;  
  
ราชา สโงฺฆ สนฺนิปติ, มริจวฎฺฎิมเห วิยฯ  
</li>
  
<li>
ปุถุชฺชนา’ว อฎฺฐํสุ, ภิกฺขู ปฐมภูมิยํ;  
  
เตปิฎกา ทุติยาย, โสตาปนฺนาทโย ปนฯ  
</li>
  
<li>
เอเก  
เกเยว อฎฺฐํสุ, ตติยาทีสุ ภูมิสุ;  
  
อรหโนฺต จ อฎฺฐํสุ, อุทฺธํ จตูสุ ภูมิสุฯ  
</li>
  
<li>
สงฺฆสฺส ทตฺวา ปาสาทํ, ทกฺขิณมฺพุปุรสฺสรํ;  
  
ราชา’ทตฺถ มหาทานํ, สตฺตาหํ ปุพฺพกํ ปิยฯ  
</li>
  
<li>
ปาสาท มหจตฺตานิ, มหาจาเคน ราชินา;  
  
อนคฺฆานิ ฐเปตฺวาน, อเหสุํ ติํสโกฎิโยฯ  
</li>
  
<li>
นิสฺสาเร ธนนิจเย วิเสสสารํ,  
  
เย ทานํ ปริคณยนฺติ สาธุปญฺญา;  
  
เต ทานํ วิปุลมเปต จิตฺตสงฺคา,  
  
สตฺตานํ หิตปรมา ททนฺติ เอวํติฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
โลกหปาสาทมโห นาม  
</li>
  
สตฺตวีสติโม ปริเจฺฉโทฯ  
</li>

<h2>ขุทฺทกนิกาเย</h2>
<h2>จริยาปิฎกปาฬิ</h2>
<h1>๑. อกิตฺติวโคฺค</h1>
<h3>๑. อกิตฺติจริยา</h3>
<ol>
<li>
‘‘กเปฺป  
  
  
  
จ สตสหเสฺส, จตุโร จ อสงฺขิเย;  
  
เอตฺถนฺตเร ยํ จริตํ, สพฺพํ ตํ โพธิปาจนํฯ  
</li>
  
<li>
‘‘อตีตกเปฺป จริตํ, ฐปยิตฺวา ภวาภเว;  
  
อิมมฺหิ กเปฺป จริตํ, ปวกฺขิสฺสํ สุโณหิ เมฯ  
</li>
  
<li>
‘‘ยทา อหํ พฺรหารเญฺญ, สุเญฺญ วิปินกานเน;  
  
อโชฺฌคาเหตฺวา  
อโชฺฌคเหตฺวา (สี. สฺยา.)  
วิหรามิ, อกิตฺติ นาม ตาปโสฯ  
</li>
  
<li>
‘‘ตทา  
มํ ตปเตเชน, สนฺตโตฺต ติทิวาภิภู;  
  
ธาเรโนฺต พฺราหฺมณวณฺณํ, ภิกฺขาย มํ อุปาคมิฯ  
</li>
  
<li>
‘‘ปวนา อาภตํ ปณฺณํ, อเตลญฺจ อโลณิกํ;  
  
มม ทฺวาเร ฐิตํ ทิสฺวา, สกฎาเหน อากิริํฯ  
</li>
  
<li>
‘‘ตสฺส ทตฺวานหํ ปณฺณํ, นิกฺกุชฺชิตฺวาน ภาชนํ;  
  
ปุเนสนํ ชหิตฺวาน, ปาวิสิํ ปณฺณสาลกํฯ  
</li>
  
<li>
‘‘ทุติยมฺปิ  
ตติยมฺปิ, อุปคญฺฉิ มมนฺติกํ;  
  
อกมฺปิโต อโนลโคฺค, เอวเมวมทาสหํฯ  
</li>
  
<li>
‘‘น  
เม ตปฺปจฺจยา อตฺถิ, สรีรสฺมิํ วิวณฺณิยํ;  
  
ปีติสุเขน รติยา, วีตินาเมมิ ตํ ทิวํฯ  
</li>
  
<li>
‘‘ยทิ มาสมฺปิ เทฺวมาสํ, ทกฺขิเณยฺยํ วรํ ลเภ;  
  
อกมฺปิโต อโนลีโน, ทเทยฺยํ ทานมุตฺตมํฯ  
</li>
  
<li>
‘‘น ตสฺส ทานํ ททมาโน, ยสํ ลาภญฺจ ปตฺถยิํ;  
  
สพฺพญฺญุตํ ปตฺถยาโน, ตานิ กมฺมานิ อาจริ’’นฺติฯ  
</li>
  
อกิตฺติจริยํ ปฐมํฯ  
</li>
  
<h3>๒. สงฺขจริยา</h3>
</ol>
<ol>
<li>
‘‘ปุนาปรํ  
ยทา โหมิ, พฺราหฺมโณ สงฺขสวฺหโย;  
  
มหาสมุทฺทํ ตริตุกาโม, อุปคจฺฉามิ ปฎฺฎนํฯ  
</li>
  
<li>
‘‘ตตฺถทฺทสํ ปฎิปเถ, สยมฺภุํ อปราชิตํ;  
  
กนฺตารทฺธานํ ปฎิปนฺนํ  
กนฺตารทฺธานปฎิปนฺนํ (สี. สฺยา.)  
, ตตฺตาย กฐินภูมิยาฯ  
</li>
  
<li>
‘‘ตมหํ ปฎิปเถ ทิสฺวา, อิมมตฺถํ วิจินฺตยิํ;  
  
‘อิทํ เขตฺตํ อนุปฺปตฺตํ, ปุญฺญกามสฺส ชนฺตุโนฯ  
</li>
  
<li>
‘‘‘ยถา กสฺสโก ปุริโส, เขตฺตํ ทิสฺวา มหาคมํ;  
  
ตตฺถ พีชํ น โรเปติ, น โส ธเญฺญน อตฺถิโกฯ  
</li>
  
<li>
‘‘‘เอวเมวาหํ  
  
ปุญฺญกาโม, ทิสฺวา เขตฺตวรุตฺตมํ;  
  
ยทิ ตตฺถ การํ น กโรมิ, นาหํ ปุเญฺญน อตฺถิโกฯ  
</li>
  
<li>
‘‘‘ยถา อมโจฺจ มุทฺทิกาโม, รโญฺญ อเนฺตปุเร ชเน;  
  
น เทติ เตสํ ธนธญฺญํ, มุทฺทิโต ปริหายติฯ  
</li>
  
<li>
‘‘‘เอวเมวาหํ ปุญฺญกาโม, วิปุลํ ทิสฺวาน ทกฺขิณํ;  
  
ยทิ ตสฺส ทานํ น ททามิ, ปริหายิสฺสามิ ปุญฺญโต’ฯ  
</li>
  
<li>
‘‘เอวาหํ จินฺตยิตฺวาน, โอโรหิตฺวา อุปาหนา;  
  
ตสฺส ปาทานิ วนฺทิตฺวา, อทาสิํ ฉตฺตุปาหนํฯ  
</li>
  
<li>
‘‘เตเนวาหํ  
สตคุณโต, สุขุมาโล สุเขธิโต;  
  
อปิ จ ทานํ ปริปูเรโนฺต, เอวํ ตสฺส อทาสห’’นฺติฯ  
</li>
  
สงฺขจริยํ ทุติยํฯ  
</li>
  
<h3>๓. กุรุราชจริยา</h3>
</ol>
<ol>
<li>
‘‘ปุนาปรํ ยทา โหมิ, อินฺทปเตฺถ  
อินฺทปเตฺต (สี. ก.)  
ปุรุตฺตเม;  
  
ราชา ธนญฺจโย นาม, กุสเล ทสหุปาคโตฯ  
</li>
  
<li>
‘‘กลิงฺครฎฺฐวิสยา, พฺราหฺมณา อุปคญฺฉุ มํ;  
  
อายาจุํ มํ หตฺถินาคํ, ธญฺญํ มงฺคลสมฺมตํฯ  
</li>
  
<li>
‘‘‘อวุฎฺฐิโก ชนปโท, ทุพฺภิโกฺข ฉาตโก มหา;  
  
ททาหิ ปวรํ นาคํ, นีลํ อญฺชนสวฺหยํฯ  
</li>
  
<li>
‘‘‘น  
  
เม ยาจกมนุปฺปเตฺต, ปฎิเกฺขโป อนุจฺฉโว;  
  
มา เม ภิชฺชิ สมาทานํ, ทสฺสามิ วิปุลํ คชํ’ฯ  
</li>
  
<li>
‘‘นาคํ คเหตฺวา โสณฺฑาย, ภิงฺคาเร  
ภิงฺกาเร (สี.)  
รตนามเย;  
  
ชลํ หเตฺถ อากิริตฺวา, พฺราหฺมณานํ อทํ คชํฯ  
</li>
  
<li>
‘‘ตสฺส นาเค ปทินฺนมฺหิ, อมจฺจา เอตทพฺรวุํ;  
  
‘กิํ นุ ตุยฺหํ วรํ นาคํ, ยาจกานํ ปทสฺสสิฯ  
</li>
  
<li>
‘‘‘ธญฺญํ  
มงฺคลสมฺปนฺนํ, สงฺคามวิชยุตฺตมํ;  
  
ตสฺมิํ นาเค ปทินฺนมฺหิ, กิํ เต รชฺชํ กริสฺสติฯ  
</li>
  
<li>
‘‘‘รชฺชมฺปิ เม ทเท สพฺพํ, สรีรํ ทชฺชมตฺตโน;  
  
สพฺพญฺญุตํ ปิยํ มยฺหํ, ตสฺมา นาคํ อทาสห’’’นฺติฯ  
</li>
  
กุรุราชจริยํ ตติยํฯ  
</li>
  
<h3>๔. มหาสุทสฺสนจริยา</h3>
</ol>
<ol>
<li>
‘‘กุสาวติมฺหิ นคเร, ยทา อาสิํ มหีปติ;  
  
มหาสุทสฺสโน นาม, จกฺกวตฺตี มหพฺพโลฯ  
</li>
  
<li>
‘‘ตตฺถาหํ  
ทิวเส ติกฺขตฺตุํ, โฆสาเปมิ ตหิํ ตหิํ;  
  
‘โก กิํ อิจฺฉติ ปเตฺถติ, กสฺส กิํ ทียตู ธนํฯ  
</li>
  
<li>
‘‘‘โก ฉาตโก โก ตสิโต, โก มาลํ โก วิเลปนํ;  
  
นานารตฺตานิ วตฺถานิ, โก นโคฺค ปริทหิสฺสติฯ  
</li>
  
<li>
‘‘‘โก ปเถ ฉตฺตมาเทติ, โกปาหนา มุทู สุภา’;  
  
อิติ  
สายญฺจ ปาโต จ, โฆสาเปมิ ตหิํ ตหิํฯ  
</li>
  
<li>
‘‘น ตํ ทสสุ ฐาเนสุ, นปิ ฐานสเตสุ วา;  
  
อเนกสตฐาเนสุ, ปฎิยตฺตํ ยาจเก ธนํฯ  
</li>
  
<li>
‘‘ทิวา วา ยทิ วา รตฺติํ, ยทิ เอติ วนิพฺพโก;  
  
ลทฺธา ยทิจฺฉกํ โภคํ, ปูรหโตฺถว คจฺฉติฯ  
</li>
  
<li>
‘‘เอวรูปํ มหาทานํ, อทาสิํ ยาวชีวิกํ;  
  
นปาหํ เทสฺสํ ธนํ ทมฺมิ, นปิ นตฺถิ นิจโย มยิฯ  
</li>
  
<li>
‘‘ยถาปิ อาตุโร นาม, โรคโต ปริมุตฺติยา;  
  
ธเนน เวชฺชํ ตเปฺปตฺวา, โรคโต ปริมุจฺจติฯ  
</li>
  
<li>
‘‘ตเถวาหํ ชานมาโน, ปริปูเรตุมเสสโต;  
  
อูนมนํ ปูรยิตุํ, เทมิ ทานํ วนิพฺพเก;  
  
นิราลโย อปจฺจาโส, สโมฺพธิมนุปตฺติยา’’ติฯ  
</li>
  
มหาสุทสฺสนจริยํ จตุตฺถํฯ  
</li>
  
<h3>๕. มหาโควินฺทจริยา</h3>
</ol>
<ol>
<li>
‘‘ปุนาปรํ  
  
ยทา โหมิ, สตฺตราชปุโรหิโต;  
  
ปูชิโต นรเทเวหิ, มหาโควินฺทพฺราหฺมโณฯ  
</li>
  
<li>
‘‘ตทาหํ สตฺตรเชฺชสุ, ยํ เม อาสิ อุปายนํ;  
  
เตน เทมิ มหาทานํ, อโกฺขพฺภํ  
อโกฺขภํ (สฺยา. กํ.)  
สาครูปมํฯ  
</li>
  
<li>
‘‘น  
  
เม เทสฺสํ ธนํ ธญฺญํ, นปิ นตฺถิ นิจโย มยิ;  
  
สพฺพญฺญุตํ ปิยํ มยฺหํ, ตสฺมา เทมิ วรํ ธน’’นฺติฯ  
</li>
  
มหาโควินฺทจริยํ ปญฺจมํฯ  
</li>
  
<h3>๖. นิมิราชจริยา</h3>
</ol>
<ol>
<li>
‘‘ปุนาปรํ ยทา โหมิ, มิถิลายํ ปุรุตฺตเม;  
  
นิมิ นาม มหาราชา, ปณฺฑิโต กุสลตฺถิโกฯ  
</li>
  
<li>
‘‘ตทาหํ มาปยิตฺวาน, จตุสฺสาลํ จตุมฺมุขํ;  
  
ตตฺถ ทานํ ปวเตฺตสิํ, มิคปกฺขินราทินํฯ  
</li>
  
<li>
‘‘อจฺฉาทนญฺจ สยนํ, อนฺนํ ปานญฺจ โภชนํ;  
  
อโพฺพจฺฉินฺนํ กริตฺวาน, มหาทานํ ปวตฺตยิํฯ  
</li>
  
<li>
‘‘ยถาปิ เสวโก สามิํ, ธนเหตุมุปาคโต;  
  
กาเยน วาจา มนสา, อาราธนียเมสติฯ  
</li>
  
<li>
‘‘ตเถวาหํ สพฺพภเว, ปริเยสิสฺสามิ โพธิชํ;  
  
ทาเนน สเตฺต ตเปฺปตฺวา, อิจฺฉามิ โพธิมุตฺตม’’นฺติฯ  
</li>
  
นิมิราชจริยํ ฉฎฺฐํฯ  
</li>
  
<h3>๗. จนฺทกุมารจริยา</h3>
</ol>
<ol>
<li>
‘‘ปุนาปรํ  
  
ยทา โหมิ, เอกราชสฺส อตฺรโช;  
  
นคเร ปุปฺผวติยา, กุมาโร จนฺทสวฺหโยฯ  
</li>
  
<li>
‘‘ตทาหํ  
ยชนา มุโตฺต, นิกฺขโนฺต ยญฺญวาฎโต;  
  
สํเวคํ ชนยิตฺวาน, มหาทานํ ปวตฺตยิํฯ  
</li>
  
<li>
‘‘นาหํ ปิวามิ ขาทามิ, นปิ ภุญฺชามิ โภชนํ;  
  
ทกฺขิเณเยฺย อทตฺวาน, อปิ ฉปฺปญฺจรตฺติโยฯ  
</li>
  
<li>
‘‘ยถาปิ  
วาณิโช นาม, กตฺวาน ภณฺฑสญฺจยํ;  
  
ยตฺถ ลาโภ มหา โหติ, ตตฺถ ตํ  
ตตฺถ นํ (สี.), ตตฺถ (ก.)  
หรติ ภณฺฑกํฯ  
</li>
  
<li>
‘‘ตเถว สกภุตฺตาปิ, ปเร ทินฺนํ มหปฺผลํ;  
  
ตสฺมา ปรสฺส ทาตพฺพํ, สตภาโค ภวิสฺสติฯ  
</li>
  
<li>
‘‘เอตมตฺถวสํ ญตฺวา, เทมิ ทานํ ภวาภเว;  
  
น ปฎิกฺกมามิ ทานโต, สโมฺพธิมนุปตฺติยา’’ติฯ  
</li>
  
จนฺทกุมารจริยํ สตฺตมํฯ  
</li>
  
<h3>๘. สิวิราชจริยา</h3>
</ol>
<ol>
<li>
‘‘อริฎฺฐสวฺหเย นคเร, สิวินามาสิ ขตฺติโย;  
  
นิสชฺช ปาสาทวเร, เอวํ จิเนฺตสหํ ตทาฯ  
</li>
  
<li>
‘‘‘ยํ กิญฺจิ มานุสํ ทานํ, อทินฺนํ เม น วิชฺชติ;  
  
โยปิ ยาเจยฺย มํ จกฺขุํ, ทเทยฺยํ อวิกมฺปิโต’ฯ  
</li>
  
<li>
‘‘มม สงฺกปฺปมญฺญาย, สโกฺก เทวานมิสฺสโร;  
  
นิสิโนฺน เทวปริสาย, อิทํ วจนมพฺรวิฯ  
</li>
  
<li>
‘‘‘นิสชฺช  
ปาสาทวเร, สิวิราชา มหิทฺธิโก;  
  
จิเนฺตโนฺต วิวิธํ ทานํ, อเทยฺยํ โส น ปสฺสติฯ  
</li>
  
<li>
‘‘‘ตถํ  
นุ วิตถํ เนตํ, หนฺท วีมํสยามิ ตํ;  
  
มุหุตฺตํ อาคเมยฺยาถ, ยาว ชานามิ ตํ มนํ’ฯ  
</li>
  
<li>
‘‘ปเวธมาโน ปลิตสิโร, วลิคโตฺต  
วลิตคโตฺต (สี.)  
ชราตุโร;  
  
อนฺธวโณฺณว หุตฺวาน, ราชานํ อุปสงฺกมิฯ  
</li>
  
<li>
‘‘โส  
ตทา ปคฺคเหตฺวาน, วามํ ทกฺขิณพาหุ จ;  
  
สิรสฺมิํ อญฺชลิํ กตฺวา, อิทํ วจนมพฺรวิฯ  
</li>
  
<li>
‘‘‘ยาจามิ ตํ มหาราช, ธมฺมิก รฎฺฐวฑฺฒน;  
  
ตว ทานรตา กิตฺติ, อุคฺคตา เทวมานุเสฯ  
</li>
  
<li>
‘‘‘อุโภปิ  
เนตฺตา นยนา, อนฺธา อุปหตา มม;  
  
เอกํ เม นยนํ เทหิ, ตฺวมฺปิ เอเกน ยาปย’ฯ

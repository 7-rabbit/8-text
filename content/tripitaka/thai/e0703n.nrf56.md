<h1>สตฺตปญฺญาสติม ปริเจฺฉท</h1>
<h3>สํคหกรโณนาม</h3>
<ol>
<li>
ลงฺการกฺขาย สจิเว, พลิโน โยธสมฺมเต;  
  
ปฎิปาฎิํ สมุทฺทิสฺส, สมนฺตา สนฺนิโยชยิฯ  
</li>
  
<li>
อภิเสกมงฺคลตฺถาย, ปาสาทาทิมเนกตํ;  
  
กิจฺจํ สมฺปาทนียนฺติ, สจิวํ สนฺนิโยชิยฯ  
</li>
  
<li>
วนฺทนีเย’ภิวนฺทโนฺต, ปเทเสเนกเกตหิํ;  
  
เนตฺวา มาสตฺตยํ คญฺฉิ, ปุลตฺถินครํ ปุนฯ  
</li>
  
<li>
วิสฺสุโต อาทิมลย-นาเมน พลนายโก;  
  
อุชุปจฺจตฺถิโก หุตฺวา, มหีปาลสฺส สพฺพโสฯ  
</li>
  
<li>
คงฺคา มตฺถยุ’ปาคญฺฉิ, พลํ สพฺพํ สมาทิย;  
  
อนฺทูติ วิสฺสุตํ มนฺท-ปโญฺญ คามํ ปุรนฺติเกฯ  
</li>
  
<li>
ลํกิสฺสโร ตหิํ คนฺตฺวา, อุทฺธริตฺวา ตมุทฺธตํ;  
  
ปุลตฺถินครํ คญฺฉิ, วเส วตฺติย ตํ พลํฯ  
</li>
  
<li>
ยุวราชปทํเยว, สิโต สโนฺต ลิขาปยิ;  
  
โส สตฺตรสวสฺสานิ, สปฺปโญฺญ นรสตฺตโมฯ  
</li>
  
<li>
ตโต’นุราธนคร-มภิคมฺม  
ยถาวิธิํ;  
  
อนุโภตฺวา วิธานญฺญู, อภิเสกมหุสฺสวํฯ  
</li>
  
<li>
อฎฺฐิโต ปาปธเมฺมสุ, สุฎฺฐิโต เสฎฺฐกมฺมนิ;  
  
โส อฎฺฐารสมํ วสฺสํ, ลิขาปยิ สุสณฺฐิโตฯ  
</li>
  
<li>
ตโต อาคมฺม นิวสิ, ปุลตฺถินคเร วเร;  
  
โส สิรีสงฺฆโพธี’ติ, นามเธเยฺยน วิสฺสุโตฯ  
</li>
  
<li>
อนุชํ โส วีรพาหุ-โมปรเชฺช นิเวสิย;  
  
ทตฺวาน ทกฺขิณํ เทสํ, ตํ สงฺคณฺหิ ยถาวิธิํฯ  
</li>
  
<li>
กณิฎฺฐสฺสาถ ภาตุสฺส, ชยพาหุสฺส ภูภุโช;  
  
อาทิปาทปทํ ทตฺวา, รฎฺฐํ จา’ทาสิ โรหณํฯ  
</li>
  
<li>
ฐานนฺตรานิ สเพฺพส-มมจฺจานํ ยถารหํ;  
  
ทตฺวา รเชฺช ยถาญายํ, กรํ โยเชสิ คณฺหิตุํฯ  
</li>
  
<li>
จิรสฺสํ ปริหีนํ โส, ทยาวาโส มหีปติ;  
  
ปวเตฺตสิ ยถาธมฺมํ, ฐิตธโมฺม วินิจฺฉยํฯ  
</li>
  
<li>
เอวํ สมุทฺธฎาเนก, ริปุกณฺฎกสญฺจเย;  
  
นิจฺจํ รชฺชํ ปสาเสเนฺต, ลงฺกํสมฺมานริสฺสเรฯ  
</li>
  
<li>
ฉตฺตคฺคาหกนาโถ จ, ธมฺมเคหกนายโก;  
  
ตเถว เสฎฺฐินาโถ จ, อิเจฺจเต ภาตโร ตโยฯ  
</li>
  
<li>
รโญฺญ วิโรธิตํ ยาตา, ผลาตา ชมฺปุทีปกํ;  
  
ลงฺกํ วีสติเม วเสฺส, เอเกนูเน สโมตรุํฯ  
</li>
  
<li>
เต สเพฺพ โรหณํ รฎฺฐํ, สพฺพํ มลยมณฺฑลํ;  
  
สพฺพํ ทกฺขิณปสฺสญฺจ, สหสา ปริวตฺตยุํฯ  
</li>
  
<li>
นิปุโณ โรหณํ คนฺตฺวา, ตถา มลยมณฺฑลํ;  
  
นิฆาเตโนฺต พหู ตตฺถ, ตตฺถ ปจฺจตฺถิเก ชเนฯ  
</li>
  
<li>
สมฺมาวุปสเมตฺวา ตํ, ฐเปตฺวา สจิเว ตหิํ;  
  
ทกฺขิโณ ทกฺขิณํ เทสํ, สยํ คนฺตฺวา มหพฺพโลฯ  
</li>
  
<li>
เปเสตฺวา สมณีภาตุ-วํสชํ สจิวํ ตทา;  
  
คาเหตฺวา สมเรโฆเร, วีโร เต สกเวริโนฯ  
</li>
  
<li>
สมาโรปิยสูลมฺหิ, ลงฺกํ วิคตกณฺฎกํ;  
  
กาเรตฺวาน นิราตงฺกํ, ปุลตฺถิปุรมาคมิฯ  
</li>
  
<li>
วสนฺตี  
โจฬวิสเย, ชคติปาลราชินี;  
  
โจฬหตฺถา ปมุญฺจิตฺวา, สทฺธิํ ธีตุ กุมาริยาฯ  
</li>
  
<li>
ลีลาวตฺยาภิธานาย, นาวามารุยฺห เวคสา;  
  
ลํกาทีปมฺหิ โอติณฺณา, ปสฺสิ ลํกิสฺสรํ ตทาฯ  
</li>
  
<li>
สุตฺวา วํสกฺกมํ ตสฺสา, โส ญตฺวา สุทฺธวํสตํ;  
  
ลีลาวติํ มหิสิเตฺต, อภิสิญฺจิ นริสฺสโรฯ  
</li>
  
<li>
สา ตํ ปฎิจฺจ ราชานํ, มเหสี ธีตรํ ลภิ;  
  
นามํ ยโสธราติ’สฺสา, อกาสิ ธรณี ปติฯ  
</li>
  
<li>
เมรุกนฺทรรเฎฺฐน, สทฺธิํ ราชา สธีตรํ;  
  
วีรวมฺมสฺส ปาทาสิ, สา ลภิ ธีตโร ทุเวฯ  
</li>
  
<li>
สมานนามิกา เชฎฺฐา, สา มาตา มหิยา อหุ;  
  
สุคลา นามิกา อาสิ, ตาสุ ทฺวีสุ กณิฎฺฐิกาฯ  
</li>
  
<li>
กาลิงฺคธรณีปาล-วํสชํ จารุทสฺสนํ;  
  
ติโลกสุนฺทรํ นาม, สุกุมารํ กุมาริกํฯ  
</li>
  
<li>
กาลิงฺครฎฺฐโต ราชา, อาณาเปตฺวา จิรฎฺฐิติํ;  
  
นิชวํสสฺส อิจฺฉโนฺต, มเหสิเตฺต’ภิ เสจยิฯ  
</li>
  
<li>
สุภทฺทา จ สุมิตฺตา จ, โลกนาถวฺหยาปิ จ;  
  
รตนาวลี รูปวตี, อิติมา ปญฺจ ธีตโรฯ  
</li>
  
<li>
ปุตฺตํ วิกฺกมพาหุญฺจ, สา ลภี ธญฺญลกฺขณํ;  
  
สมฺปตฺตา สา ปชวุทฺธิํ, หรนฺติ ราชิโน มตํฯ  
</li>
  
<li>
อิตฺถาคาเรสุ เสเสสุ, วีตา สมกุลงฺคนา;  
  
คโพฺภ ชาตุมหีปาลํ, ตํ ปฎิจฺจ น สณฺฐหิฯ  
</li>
  
<li>
อเถกทิวสํ ราชา, อมจฺจคณมชฺฌโค;  
  
ปิโลกิยฐิภา สพฺพา, ธีตโร ปฎิปาฎิยาฯ  
</li>
  
<li>
ธีตุ นวมวเสสานํ, ฐเปตฺวา รตนาวลิํ;  
  
ธญฺญลกฺขณสมฺปนฺน-ปุตฺตสฺสุปฺปตฺติสุจกํฯ  
</li>
  
<li>
ลกฺขณํ ลกฺขณญฺญู โส, อปสฺสํ เปมเจควา;  
  
รตนาวลิมาหูย, ตสฺสา มุทฺธนิ จุมฺพิยฯ  
</li>
  
<li>
เตโชคุเณหิ  
จาเคหิ, ธิยา สูรตฺตเนน จ;  
  
ภูเต จ ภาวิโน เจว, สเพฺพ ภูเป’ติสายิโนฯ  
</li>
  
<li>
นิจฺจํ ลงฺกํ นิราสงฺก-เมกจฺฉตฺตกเมว จ;  
  
ปวิธาตุํ สมตฺถสฺส, สมฺมาสาสนตายิโนฯ  
</li>
  
<li>
โสภนาเนกวตฺตสฺส, อิมิสฺสา กุจฺฉิเหสฺสติ;  
  
ปุตฺตสฺสุปฺปตฺติฐานนฺติ, มุทุโต โส สมพฺรวิฯ  
</li>
  
<li>
ยาวนฺตสฺสาปิโขโจฬ-มหีปาลสฺส เนกโส;  
  
กุลาภิมานี ราชา โส, อทตฺวาน กณียสิํฯ  
</li>
  
<li>
อาณาเปตฺวา ปณฺฑุราชํ, วิสุทฺธตฺวยสมฺภวํ;  
  
อนุชํ ราชินิํ วสฺส, มิตฺตวฺหยมทาสิโสฯ  
</li>
  
<li>
สามาณภรณํ กิตฺติ, สิรี เมฆาภิธานกํ;  
  
สิริวลฺลภนามญฺจ, ชเนสิ ตนเย ตโยฯ  
</li>
  
<li>
สุภทฺท วีรพาหุสฺส, สุมิตฺตํ ชยพาหุโน;  
  
มหตา ปริหาเรน, ปาทาสิ ธรณี ปติฯ  
</li>
  
<li>
อทาสิ มาณาภรเณ, ธีตรํ รตนาวลิํ;  
  
โลกนาถวฺหยํ กิตฺติ สิริเมฆสฺส’ทาสิโสฯ  
</li>
  
<li>
รูปตฺยภิธานาย, ธีตุโย ปรตายหิ;  
  
สรีริวลฺลภสาทา, สุคลวฺห กุมาริกํฯ  
</li>
  
<li>
มธุกณฺณวภีมราช, พลกฺการ สนามเก;  
  
มเหสี พนฺธเวราช-ปุเตฺต สีหปุราคเตฯ  
</li>
  
<li>
ปสฺสิตฺวาน ปหีปาโล, ตทา สญฺชาตปีติโก;  
  
เตสํ ปาทาสิ ปเจฺจกํ, วุตฺติํ โส อนุรูปกํฯ  
</li>
  
<li>
เต สเพฺพ ลทฺธสกฺการ สมฺมานาธรณีปติํ;  
  
อาราธยนฺตา สสตํ, นิวสิํสุ ยถารุจิํฯ  
</li>
  
<li>
เอเตสํ ราชปุตฺตานํ, สุนฺทริวฺหํ กณิฎฺฐกํ;  
  
อทา วิกฺกมพาหุสฺส, นิชวํสฐิตฎฺฐิโกฯ  
</li>
  
<li>
ภิโยฺย  
วิกฺกมพาหุสฺส, ตโต ลีลาวติํ สติํ;  
  
สห โภเคน ปาทาสิ, ตทา พนฺธุ หิเตรโตฯ  
</li>
  
<li>
วิธาย เอวํ สชเน ชนิโนฺท;  
  
นิเสฺสสโต โภคสมปฺปิโต;  
  
ทยาปโร ญาติชนาตมตฺถํ;  
  
สมาจรี นีติปถานุรูปํฯ  
</li>
  
สุชนปฺปสาทสํเวคตฺถาย กเต มหาวํเส  
</li>
  
สํคหกรโณ นาม  
</li>
  
สตฺตปญฺญาสติโม ปริเจฺฉโท  
</li>

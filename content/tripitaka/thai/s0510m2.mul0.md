<h2>ขุทฺทกนิกาเย</h2>
<h2>เถราปทานปาฬิ</h2>

<h3>๑. สกิํสมฺมชฺชกเตฺถรอปทานํ</h3>
<ol>
<li>
‘‘วิปสฺสิโน  
  
  
ภควโต, ปาฎลิํ โพธิมุตฺตมํ;  
  
ทิสฺวาว ตํ ปาทปคฺคํ, ตตฺถ จิตฺตํ ปสาทยิํฯ  
</li>
  
<li>
‘‘สมฺมชฺชนิํ คเหตฺวาน, โพธิํ สมฺมชฺชิ ตาวเท;  
  
สมฺมชฺชิตฺวาน ตํ โพธิํ, อวนฺทิํ ปาฎลิํ อหํฯ  
</li>
  
<li>
‘‘ตตฺถ จิตฺตํ ปสาเทตฺวา, สิเร กตฺวาน อญฺชลิํ;  
  
นมสฺสมาโน ตํ โพธิํ, คญฺฉิํ ปฎิกุฎิํ อหํฯ  
</li>
  
<li>
‘‘ตาทิมเคฺคน  
คจฺฉามิ, สรโนฺต โพธิมุตฺตมํ;  
  
อชคโร มํ ปีเฬสิ, โฆรรูโป มหพฺพโลฯ  
</li>
  
<li>
‘‘อาสเนฺน  
เม กตํ กมฺมํ, ผเลน โตสยี มมํ;  
  
กเฬวรํ เม คิลติ, เทวโลเก รมามหํฯ  
</li>
  
<li>
‘‘อนาวิลํ มม จิตฺตํ, วิสุทฺธํ ปณฺฑรํ สทา;  
  
โสกสลฺลํ น ชานามิ, จิตฺตสนฺตาปนํ มมฯ  
</li>
  
<li>
‘‘กุฎฺฐํ คโณฺฑ กิลาโส จ, อปมาโร วิตจฺฉิกา;

สมฺมชฺชเน อิทํ (สี.)  
ฯ  
</li>
  
<li>
‘‘โสโก  
จ ปริเทโว จ, หทเย เม น วิชฺชติ;

</li>
  
<li>
‘‘สมาธีสุ  
น มชฺชามิ  
สมาธีสุ น สชฺชามิ (สี.), สมาธิํ ปุน ปชฺชามิ (สฺยา)  
, วิสทํ โหติ มานสํ;  
  
ยํ ยํ สมาธิมิจฺฉามิ, โส โส สมฺปชฺชเต มมํฯ  
</li>
  
<li>
‘‘รชนีเย น รชฺชามิ, อโถ ทุสฺสนิเยสุ  
โทสนิเยสุ (สี. สฺยา. ก.)  
จ;  
  
โมหนีเย น มุยฺหามิ, ผลํ สมฺมชฺชนายิทํฯ  
</li>
  
<li>
‘‘เอกนวุติโต  
เอกนวุเต อิโต (สี. สฺยา.)  
กเปฺป, ยํ กมฺมมกริํ ตทา;  
  
ทุคฺคติํ นาภิชานามิ, ผลํ สมฺมชฺชนายิทํฯ  
</li>
  
<li>
‘‘กิเลสา ฌาปิตา มยฺหํ, ภวา สเพฺพ สมูหตา;  
  
นาโคว พนฺธนํ เฉตฺวา, วิหรามิ อนาสโวฯ  
</li>
  
<li>
‘‘สฺวาคตํ วต เม อาสิ, มม พุทฺธสฺส สนฺติเก;  
  
ติโสฺส วิชฺชา อนุปฺปตฺตา, กตํ พุทฺธสฺส สาสนํฯ  
</li>
  
<li>
‘‘ปฎิสมฺภิทา จตโสฺส, วิโมกฺขาปิ จ อฎฺฐิเม;  
  
ฉฬภิญฺญา สจฺฉิกตา, กตํ พุทฺธสฺส สาสนํ’’ฯ  
</li>
  
อิตฺถํ สุทํ อายสฺมา สกิํสมฺมชฺชโก เถโร อิมา คาถาโย  
</li>

</li>
  
<li>
‘‘ปทุมุตฺตโร นาม ชิโน, สพฺพธมฺมาน ปารคู;

</li>
  
<li>

</li>
  
<li>
‘‘‘อิทํ

</li>
  
<li>
‘‘เอวาหํ จินฺตยิตฺวาน, สกํ จิตฺตํ ปสาทยิํ;

</li>
  
<li>
‘‘เอกํ  
ทุสฺสํ ททิตฺวาน, อุกฺกุฎฺฐิํ สมฺปวตฺตยิํ;  
  
‘ยทิ พุโทฺธ ตุวํ วีร, ตาเรหิ มํ มหามุนิ’ฯ  
</li>
  
<li>
‘‘ปทุมุตฺตโร โลกวิทู, อาหุตีนํ ปฎิคฺคโห;  
  
มม ทานํ ปกิเตฺตโนฺต, อกา เม อนุโมทนํฯ  
</li>
  
<li>
‘‘‘อิมินา เอกทุเสฺสน, เจตนาปณิธีหิ จ;

</li>
  
<li>
‘‘‘ฉตฺติํสกฺขตฺตุํ เทวิโนฺท, เทวรชฺชํ กริสฺสสิ;  
  
เตตฺติํสกฺขตฺตุํ ราชา จ, จกฺกวตฺตี

ภวิสฺสสิฯ  
</li>
  
<li>
‘‘‘ปเทสรชฺชํ วิปุลํ, คณนาโต อสงฺขิยํ

</li>
  
<li>

<li>

</li>
  
<li>
‘‘ปทุทฺธาเร

</li>
  
<li>

<li>

</li>
  
<li>

<li>

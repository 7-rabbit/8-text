<h1>นวม ปริเจฺฉท</h1>
<h3>อภยาภิเสโก</h3>
<ol>
<li>
มเหสี ชนยี ปุเตฺต, ทส เอกญฺจ ธีตรํ;  
  
สพฺพเชโฎฺฐ’ภโย นาม, จิตฺตานาม กนิฎฺฐิกาฯ  
</li>
  
<li>
ปสฺสิตฺวา ตํ วิยากํสุ, พฺราหฺมณา มนฺตปารคา;  
  
‘‘รชฺชเหตุ สุโต อสฺสา, ฆาตยิสฺสติ มาตุเล’’ฯ  
</li>
  
<li>
ฆาเตสฺสามิ กนิฎฺฐินฺติ, นิจฺฉิเต ภาตรา’ภโย;  
  
วาเรสิ กาเล วาเสสุํ, เคหํ ตํ เอกถุณิเกฯ  
</li>
  
<li>
รโญฺญ จ สิริคเพฺภน, ตสฺส ทฺวารมการยุํ;  
  
อโนฺต ฐเปสุํ เอกญฺจ, ทาสิํ นรสตํ พหิฯ  
</li>
  
<li>
รูเปนุ’มฺมาทยี นเร, ทิฎฺฐมตฺตาว สายโต;  
  
ตโต อุมฺมาทจิตฺตา’ติ, นามํ โสปปทํ ลภิฯ  
</li>
  
<li>
สุตฺวาน ลํกาคมนํ, ภทฺทกจฺจานเทวิยา;  
  
มาตรา โจทิตา ปุตฺตา, ฐเปเตฺว’กญฺจ อาคมุํฯ  
</li>
  
<li>
ทิสฺวาน เต ปณฺฑุวาสุ-เทวํ ลงฺกินฺทมาคตา;  
  
ทิสฺวาน ตํ กนิฎฺฐิญฺจ, โรทิตฺวา สห ตาย จฯ  
</li>
  
<li>
รญฺญา สุกตสกฺการา, รโญฺญ’นุญฺญาย จาริกํ;  
  
จริํสุ ลํกาทีปมฺหิ, นิวสุญฺจ ยถารุจิํฯ  
</li>
  
<li>
ราเมน  
วุสิตฐานํ, รามโคณนฺติ วุจฺจติ;  
  
อุรุเวลา’นุราธานํ, นิวาสา จ ตถา ตถาฯ  
</li>
  
<li>
ตถา วิชิตทีฆายุ-โรหณานํ นิวาสกา;  
  
วิชิตคาโม ทีฆายุ-โรหณนฺติ จ วุจฺจติฯ  
</li>
  
<li>
กาเรสิ อนุราโธ โส, วาปิํ ทกฺขิณโต ตโต;  
  
การาเปตฺวา ราชเคหํ, ตตฺถ วาสมกปฺปยีฯ  
</li>
  
<li>
มหาราชา ปณฺฑุวาสุ-เทโว ชฎฺฐสุตํ สกํ;  
  
อภยํ อุปรชฺชมฺหิ, กาเล สมภิเสจยิฯ  
</li>
  
<li>
ทีฆายุสฺส กุมารสฺส, ตนโย ทีฆคามณิ;  
  
สุตฺวา อุมฺมาทจิตฺตํ ตํ, ตสฺสํ ชาตกุตูหโลฯ  
</li>
  
<li>
คเหตฺวา’ปติสฺส คามํ, ตํ อปสฺสิ มนุชาธิปํ;  
  
อทา สโหปราเชน, ราชุปฎฺฐานมสฺสโสฯ  
</li>
  
<li>
ควกฺขาภิมุขฎฺฐาเน, ตํ อุเปจฺจ ฐิตํ ตุสา;  
  
ทิสฺวาน คามิณิํ จิตฺตา, รตฺตจิตฺตา’ห ทาสิกํฯ  
</li>
  
<li>
‘‘โก เอโส’’ติ ตโต สุตฺวา,  
  
มาตุลสฺส สุโตอิติ;  
  
ทาสิํ ตตฺถ นิโยเชสิ,  
  
สทฺธิํกตฺวาน โส ตโตฯ  
</li>
  
<li>
ควกฺขมฺหิ ฑสาเปตฺวา, รตฺติํ กกฺกฎยนฺตกํ;  
  
อารุยฺห ฉินฺทยิตฺวาน, กวาฎํ เตน ปาวิสิฯ  
</li>
  
<li>
ตาย สทฺธิํ วสิตฺวาน, ปจฺจูเสเยว นิกฺขมิ;  
  
เอวํ นิจฺจํ วสี ตตฺถ, ฉิทฺทาภาวา อปากโฎฯ  
</li>
  
<li>
สา เตน อคฺคหี คพฺภํ, คเพฺภ ปริณเต ตโต;  
  
มาตุ อาโรจยี ทาสี, มาตา ปุจฺฉิย ธีตรํฯ  
</li>
  
<li>
รโญฺญ อาโรจยี ราชา, อมเนฺตตฺวา สุเต’พฺรหฺมา;  
  
โปสิโยโส’ปิ อเมฺหหิ, เทว ตเสฺสว ตํ อิติฯ  
</li>
  
<li>
ปุโตฺต เจ มารยิสฺสาม, ต’นฺติ ตสฺส อทํสุ ตํ;  
  
สา สูติกาลสมฺปเตฺต, สูติ เคหญฺจ ปาวิสิํฯ  
</li>
  
<li>
สงฺกิตฺวา  
โคปกํ จิตฺตํ, กาลเวลญฺจ ทาสกํ;  
  
ตสฺมิํ กเมฺม นิสฺสายา’ติ, คามณิ ปริจารเกฯ  
</li>
  
<li>
เต ปฎิญฺญํ อเทเนฺต เต, ราชปุตฺตา อฆาตยุํ;  
  
ยกฺขา หุตฺวาน รกฺขิํสุ, อุโภ คเพฺภ กุมารกํฯ  
</li>
  
<li>
อญฺญํ อุปวิชญฺญํ สา, สลฺลกฺขา เปสิ ทาสิยา;  
  
จิตฺตา สา ชนยี ปุตฺตํ, สา อิตฺถิ ปน ธีตรํฯ  
</li>
  
<li>
จิตฺตํ สหสฺสํ ทาเปตฺวา, ตสฺสา ปุตฺตํ สกมฺปิ จ;  
  
อาณาเปตฺวา ธีตรํ ตํ, นิปชฺชาเปสิ สนฺติเกฯ  
</li>
  
<li>
ธีตา ลทฺธา’ติ สุตฺวาน, ตุฎฺฐา ราชสุตา อหุํ;  
  
มาตา จ มาตุมาตา จ, อุโภ ปน กุมารกํฯ  
</li>
  
<li>
มาตามหสฺส นามญฺจ, เชฎฺฐสฺส มาตุลสฺส จ;  
  
เอกํ กตฺวา ตมกรุํ, ปณฺฑุกาภยนามกํฯ  
</li>
  
<li>
ลํกา ปาโล ปณฺฑุวาสุ-เทโว รชฺชมการยิ;  
  
ติํส วสฺสมฺหิ ชาตมฺหิ, มโต โส ปณฺฑุกาภเยฯ  
</li>
  
<li>
ตสฺมิํ มตสฺมิํ มนุชาธิปสฺมิํ,  
  
สเพฺพ สมาคมฺม นรินฺทปุตฺตา;  
  
ตสฺสาภยสฺสาภยทสฺส ภาตุ,  
  
ราชาภิเสกํ อกรุํ อุฬาราติฯ  
</li>
  
สุชนปฺปสาท สํเวคตฺถาย กเต มหาวํเส  
</li>
  
อภยาภิเสโก นาม  
</li>
  
นวโม ปริเจฺฉโทฯ  
</li>

<h1>๒๐. วีสติโม ปริเจฺฉโท</h1>
<h3>มคฺคามคฺคญาณทสฺสนวิสุทฺธินิเทฺทโส</h3>
<ol>
<li>
กลาปสมฺมสเนเนว  
  
,  
  
โยโค กรณิโย สิยา;  
  
มคฺคามเคฺค ตุ ญาณํ ต-  
  
มธิคนฺตุํ ปนิจฺฉตาฯ  
</li>
  
<li>
ปจฺจุปฺปนฺนสฺส ธมฺมสฺส, นิพฺพตฺติ อุทโย มโต;  
  
วโย วิปริณาโมติ, ตเสฺสว สมุทีริตาฯ  
</li>
  
<li>
อนุปสฺสนาปิ  
ญาณนฺติ, วรญาเณน เทสิตํ;  
  
โส ปเนวํ ปชานาติ, โยคาวจรมาณโวฯ  
</li>
  
<li>
อิมสฺส นามรูปสฺส, ปุเพฺพ อุปฺปตฺติโต ปน;  
  
นิจโย ราสิ วา นตฺถิ, ตถา อุปฺปชฺชโตปิ จฯ  
</li>
  
<li>
ราสิโต นิจยา วาปิ, นตฺถิ อาคมนนฺติ จ;  
  
ตถา นิรุชฺฌมานสฺส, น ทิสาคมนนฺติ จฯ  
</li>
  
<li>
นิรุทฺธสฺสาปิ เอกสฺมิํ, ฐาเน นตฺถิ จโยติ จ;  
  
เอตฺถ วีณูปมา วุตฺตา, เอตสฺสตฺถสฺส ทีปเนฯ  
</li>
  
<li>
อุทพฺพยมนกฺการเมวํ สเงฺขปโต ปน;  
  
กตฺวา ตเสฺสว ญาณสฺส, วิภงฺคสฺส วเสน ตุฯ  
</li>
  
<li>
‘‘อวิชฺชาสมุทยา รูปสมุทโย’’ติ หิ อาทินา;  
  
นเยเนเกกขนฺธสฺส, อุทยพฺพยทสฺสเนฯ  
</li>
  
<li>
ทส ทสาติ กตฺวาน, วุตฺตา ปญฺญาสลกฺขณา;  
  
เตสํ ปน วเสนาปิ, ธเมฺม สมนุปสฺสติฯ  
</li>
  
<li>
เอวํ รูปุทโย โหติ, เอวมสฺส วโย อิติ;  
  
อุเทติ เอวํ รูปมฺปิ, เอวํ รูปํ ตุ เวติ จฯ  
</li>
  
<li>
เอวํ  
ปจฺจยโตเปตฺถ, ขณโต อุทยพฺพยํ;  
  
ปสฺสโต สพฺพธมฺมา จ, ปากฎา โหนฺติ ตสฺส เตฯ  
</li>
  
<li>
อุทเก ทณฺฑราชีว, อารเคฺคริว สาสโป;  
  
วิชฺชุปฺปาทาว ธมฺมา เต, ปริตฺตฎฺฐายิโน สิยุํฯ  
</li>
  
<li>
กทลีสุปินาลาตจกฺกมายุปมา อิเม;  
  
อสารา ปน นิสฺสารา, หุตฺวา ขายนฺติ โยคิโนฯ  
</li>
  
<li>
เอวเมตฺตาวตา  
เตน, อุทยพฺพยทสฺสนํ;  
  
ลกฺขณานิ จ ปญฺญาส, ปฎิวิชฺฌ ฐิตํ ปนฯ  
</li>
  
<li>
ญาณํ  
อธิคตํ โหติ, ตรุณํ ปฐมํ ปน;  
  
ยสฺส จาธิคมา โยคี, โหตารทฺธวิปสฺสโกฯ  
</li>
  
<li>
วิปสฺสนาย เหตาย,  
  
กรุณายาถ โยคิโน;  
  
วิปสฺสกสฺส ชายเนฺต,  
  
อุปเกฺลสา ทเสวิเมฯ  
</li>
  
<li>
โอภาโส ปีติ ปสฺสทฺธิ, ญาณํ สทฺธา สตี สุขํ;  
  
อุเปกฺขา วีริยํ นิกนฺตีติ, อุปเกฺลสา ทเสวิเมฯ  
</li>
  
<li>
สมฺปตฺตปฎิเวธสฺส, โสตาปนฺนาทิโนปิ จ;  
  
ตถา วิปฺปฎิปนฺนสฺส, อุปเกฺลสา น ชายเรฯ  
</li>
  
<li>
สมฺมาว ปฎิปนฺนสฺส, ยุตฺตโยคสฺส ภิกฺขุโน;  
  
สทา วิปสฺสกเสฺสว, อุปฺปชฺชนฺติ กิรสฺสุ เตฯ  
</li>
  
<li>
วิปสฺสนาย โอภาโส,  
โอภาโส  
ติ ปวุจฺจติ;  
  
ตสฺมิํ ปน สมุปฺปเนฺน, โยคาวจรภิกฺขุ โสฯ  
</li>
  
<li>
มคฺคปฺปโตฺต ผลปฺปโตฺต, อหมสฺมีติ คณฺหติ;  
  
อมคฺคํเยว มโคฺคติ, ตเสฺสวํ ปน คณฺหโตฯ  
</li>
  
<li>
เอวํ  
วิปสฺสนาวีถิ,  
  
โอกฺกนฺตา นาม โหติ สา;  
  
โอภาสเมว โส ภิกฺขุ,  
  
อสฺสาเทโนฺต นิสีทติฯ  
</li>
  
<li>
  
ปีติ  
วิปฺปสฺสนาปีติ, ตสฺส ตสฺมิํ ขเณ ปน;  
  
ตทา ปญฺจวิธา ปีติ, ชายเนฺต ขุทฺทิกาทิกาฯ  
</li>
  
<li>
วิปสฺสนาย ปสฺสทฺธิ,  
ปสฺสทฺธี  
ติ ปวุจฺจติ;  
  
โยคิโน กายจิตฺตานิ, ปสฺสทฺธาเนว โหนฺติ หิฯ  
</li>
  
<li>
ลหูนิ  
จ มุทูเนว, กมฺมญฺญาเนว โหนฺติ หิ;  
  
ปสฺสทฺธาทีหิ โส ภิกฺขุ, อนุคฺคหิตมานโสฯ  
</li>
  
<li>
อมานุสิํ รติํ นาม,  
  
อนุโภติ อนุตฺตรํ;  
  
ยํ สนฺธาย จ คาถาโย,  
  
ภาสิตา หิ มเหสินาฯ  
</li>
  
<li>
‘‘สุญฺญาคารํ ปวิฎฺฐสฺส, สนฺตจิตฺตสฺส ภิกฺขุโน;  
  
อมานุสี รติ โหติ, สมฺมา ธมฺมํ วิปสฺสโตฯ  
</li>
  
<li>
‘‘ยโต ยโต สมฺมสติ, ขนฺธานํ อุทยพฺพยํ;  
  
ลภตี ปีติปาโมชฺชํ, อมตํ ตํ วิชานต’’นฺติฯ  
</li>
  
<li>
  
ญาณาทโย  
อุปเกฺลสา, เญยฺยา วุตฺตนเยนิธ;  
  
เอเต ทส อุปเกฺลสา, วชฺชนียาว โยคินาฯ  
</li>
  
<li>
เอโตฺถภาสาทโย ธมฺมา,  
  
อุปเกฺลสสฺส วตฺถุโต;  
  
อุปเกฺลสาติ นิทฺทิฎฺฐา,  
  
อุปเกฺลสนิกนฺติ ตุฯ  
</li>
  
<li>
ตํ  
ตมาวชฺชมานสฺส, ภาวนา ปริหายติ;  
  
อสเตฺต สตฺตสญฺญี จ, โหติ อปฺปสฺสุโต นโรฯ  
</li>
  
<li>
สโพฺพภาสาทโย  
ธเมฺม, น มโคฺคติ วิจารยํ;  
  
มโคฺค วิปสฺสนาญาณํ, อิเจฺจวํ ปน ปณฺฑิโตฯ  
</li>
  
<li>
ววตฺถเปติ มคฺคญฺจ, อมคฺคเญฺจว เจตสา;  
  
ตสฺส เจวํ อยํ มโคฺค, นายํ มโคฺคติ โยคิโนฯ  
</li>
  
<li>
มคฺคามคฺคญฺจ วิญฺญาย, ฐิตญาณมิทํ ปน;  
  
มคฺคามเคฺคสุญาณ  
นฺติ, ภูริญาเณน เทสิตํฯ  
</li>
  
<li>
มคฺคามคฺคญาณทสฺสเนสุ  
โกวิทา,  
  
สาราสารเวทิโน สมาหิตาหิตา;  
  
มคฺคามคฺคญาณทสฺสนนฺติ ตํ อิทํ,  
  
พุทฺธา พุทฺธสาวกา วทนฺติ วาทิโนฯ  
</li>
  
อิติ อภิธมฺมาวตาเร มคฺคามคฺคญาณทสฺสนวิสุทฺธินิเทฺทโส  
</li>
  
นาม วีสติโม ปริเจฺฉโทฯ  
</li>

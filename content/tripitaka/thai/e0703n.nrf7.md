<h1>อฎฺฐม ปริเจฺฉท</h1>
<h3>ปณฺฑุวาสุเทวาภิเสโก</h3>
<ol>
<li>
วิชโย  
โส มหาราชา, วเสฺส อนฺติมเก ฐิโต;  
  
อิติ จินฺตยิ ‘‘วุโทฺธ’หํ, น จ วิชฺชติ เม สุโตฯ  
</li>
  
<li>
กิเจฺฉน วาสิตํ รฎฺฐํ, นเสฺสยฺย มม อจฺจเย;  
  
อาณาเปยฺยํ รชฺชเหตุ-สุมิตฺตํ ภาตรํ มมฯ  
</li>
  
<li>
อถา’มเจฺจหิ มเนฺตตฺวา, เลขํ ตตฺถ วิสชฺชยิ;  
  
เลขํ ทตฺวาน วิชโย, น จิเรน ทิวงฺคโตฯ  
</li>
  
<li>
ตสฺมิํ มเต อมจฺจา เต, เปกฺขนฺตา ขตฺติยาคมํ;  
  
อุปติสฺสคาเม ฐตฺวาน, รฎฺฐํ สมุนุสาสิสุํฯ  
</li>
  
<li>
มเต วิชยราชมฺหิ, ขตฺติยาคมนา ปุรา;  
  
เอกํ วสฺสํ อยํ ลํกา-ทีโป อาสิ อราชโกฯ  
</li>
  
<li>
ตสฺมิํ สีหปุเร ตสฺส, สีหพาหุสฺส ราชิโน;  
  
อจฺจเยน สุมิโตฺต โส, ราชา ตสฺส สุโต อหุฯ  
</li>
  
<li>
ตสฺส ปุตฺตา ตโย อาสุํ, มทฺทราชสฺส ธีตุยา;  
  
ทูตา สีหปุรํ คนฺตฺวา, รโญฺญ เลขํ อทํสุ เตฯ  
</li>
  
<li>
เลขํ สุตฺวาน โส ราชา, ปุเตฺต อามนฺตยี ตโย;  
  
อหํ มหลฺลโก ตาตา, เอโก ตุเมฺหสุ คจฺฉตุฯ  
</li>
  
<li>
ลํกํ เนกคุณํ กนฺตํ, มม ภาตุสฺส สนฺตกํ;  
  
ตสฺส’จฺจ เยน ตเตฺถว, รชฺชํ กาเรตุ โสภนํฯ  
</li>
  
<li>
กณิฎฺฐโก ปณฺฑุวาสุ-เทโว ราชกุมารโก;  
  
‘‘คมิสฺสามีติ จิเนฺตตฺวา, ญตฺวา โสตฺถิ คตมฺปิ จฯ  
</li>
  
<li>
ปิตรา สมนุญฺญาโต, ทฺวตฺติํสามจฺจ ทารเก;  
  
อาทาย อารุหี นาวํ, ปริพฺพาชกลิงฺควาฯ  
</li>
  
<li>
มหากนฺทรนชฺชา เต, มุขทฺวารมฺหิ โอตรํ;  
  
เต ปริพฺพาชเก ทิสฺวา, ชโน สกฺกริ สาธุกํฯ  
</li>
  
<li>
ปุจฺฉิตฺวา  
นครํ เอตฺถ, อุปยนฺตํ กเมน เต;  
  
อุปติสฺส คามํ สมฺปตฺตา, เทวตา ปริปาลิตาฯ  
</li>
  
<li>
อมจฺจา’นุมโต’มโจฺจ, ปุจฺฉิ เนมิตฺตกา ตหิํ;  
  
ขตฺติยา’คมนํ ตสฺส, โส พฺยากาสิ ปรํปิ จฯ  
</li>
  
<li>
สตฺตเม ทีวเสเยว, อาคมิสฺสติ ขตฺติโย;  
  
พุทฺธสาสน เมตสฺส, วํสโช’ว ฐเปสฺสติฯ  
</li>
  
<li>
สตฺตเม ทิวเสเยว, เต ปริพฺพาชเก ตหิํ;  
  
ปเตฺต ทิสฺวาน ปุจฺฉิตฺวา, อมจฺจา เต วิชานิยฯ  
</li>
  
<li>
ตํ ปณฺณุวาสุเทวํ เต, ลํการเชฺชน อปฺปยุํ;  
  
มเหสิยา อภาวาโส, น ตาว อภิเสจยิฯ  
</li>
  
<li>
อมิโตทนสกฺกสฺส, ปณฺฑุสโกฺก สุโต อหุ;  
  
ญตฺวา วินาสํ สกฺยานํ, โส อาทาย สกํ ชนํฯ  
</li>
  
<li>
คนฺตฺวา อญฺญาปเทเสน, คงฺคาปารํ ตหิํ ปุรํ;  
  
มาเปตฺวา ตตฺถ กาเรสิ, รชฺชํ ตสฺส สุเต ลภิฯ  
</li>
  
<li>
ธีตา กณิฎฺฐิตา อาสิ, ภทฺทกจฺจาน นามิกา;  
  
สพฺพลกฺขณสมฺปนฺนา, สุรูปา อภิปตฺถิตาฯ  
</li>
  
<li>
ตทตฺถํ สตฺตราชาโน, ปณฺณากาเร มหารเห;  
  
เปเสสุํ ราชิโน ตสฺส, ภีโต ราชูหิ โส ปนฯ  
</li>
  
<li>
ญตฺวาน โสตฺถิคมนํ, อภิเสกผลมฺปิ จ;  
  
สหทฺวตฺติํส อิตฺถีหิ, นาวํ อาโรปิยา’สุตํฯ  
</li>
  
<li>
คงฺคาย ขิปิ คณฺหาตุ, ปหุ เม ธีตรํ อิติ;  
  
คเหตุํ เต น สกฺขิํสุ, นาวาสาปน สีฆคาฯ  
</li>
  
<li>
ทุติเย ทิวเสเยว, โคณาคามก ปฎฺฎนํ;  
  
ปตฺตาปพฺพชิตา การา, สพฺพา ตา ตตฺถ โอตรุํฯ  
</li>
  
<li>
ปุจฺฉิตฺวา นครํ เอตฺถ, ตา กเมโนปยนฺติโย;  
  
อุปติสฺสคามํ สมฺปตฺตา, เทวตา ปริปาลิตาฯ  
</li>
  
<li>
เนมิตฺตกสฺส วจนํ, สุตฺวา ตตฺถา’คตา ตุ ตา;  
  
ทิสฺวา อมโจฺจ ปุจฺฉิตฺวา, ญตฺวา รโญฺญ สมปฺปยิฯ  
</li>
  
<li>
ตํ ปณฺฑุวาสุเทวํ เต, อมจฺจา สุทฺธพุทฺธิโน;  
  
รเชฺช สมภิสิญฺจิํสุ, ปุณฺณสพฺพมโนรถํฯ  
</li>
  
<li>
สุภทฺทกจฺจาน  
มโนมรูปินิํ;  
  
มเหสีภาเว อภิสิญฺจิย’ตฺตโน;  
  
สหคตา ตาย ปทาสิ อตฺตนา;  
  
สหาคตานํ วสิ ภูมิโป สุขนฺตีฯ  
</li>
  
สุชนปฺปสาท สํเวคตฺถาย กเต มหาวํเส  
</li>
  
ปณฺฑุวาสุเทวาภิเสโก นาม  
</li>
  
อฎฺฐโม ปริเจฺฉโทฯ  
</li>

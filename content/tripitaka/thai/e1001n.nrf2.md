<h1>ปณฺฑิตกณฺฑ</h1>
<ol>
<li>

</li>
  
<li>

อสุขสฺส กุโต ปุญฺญํ, อปุญฺญสฺส กุโต วรํฯ  
</li>
  
<li>
สุจินฺติตจินฺตี  
เจว, สุภาสิตภาสีปิ จ;

</li>
  
<li>

กตฺวา, อุตฺตตฺตํ สตฺถกญฺจนํ;  
  
ภูสนํ คชฺชปชฺชาทิํ, กโรนฺติ จ มโนหรํฯ  
</li>
  
<li>
พหุํ

</li>
  
<li>

</li>
  
<li>

ธนํ นตฺถิ, สิปฺปํ โจรา น คณฺหเร;  
  
อิธ โลเก สิปฺปํ มิตฺตํ, ปรโลเก สุขาวหํฯ  
</li>
  
<li>
ภุญฺชนตฺถํ  
กถนตฺถํ, มุขํ โหตีติ โน วเท;  
  
ยํ วาตํ วา มุขารุฬฺหํ, วจนํ ปณฺฑิโต นโรฯ  
</li>
  
<li>
ทุเมฺมเธหิ ปสํสา จ, วิญฺญูหิ ครหา จ ยา;  
  
ครหาว เสโยฺย วิญฺญูหิ, ยเญฺจ พาลปฺปสํสนาฯ  
</li>
  
<li>
อจินฺติเย

</li>

อุปเทสํ สทา คเณฺห, ครุํ สมฺมา อุปฎฺฐหํฯ  
</li>

</li>
  
<li>

สทฺธมฺมํ ปริปาเลโนฺต, สทฺธมฺมสฺมา น หายติฯ  
</li>
  
<li>

</li>
  
<li>

</li>
  
<li>

ตสฺมา มูฬฺหสหเสฺสสุ, ปโญฺญ เอโก วิเสสิยเตฯ  
</li>
  
<li>
พาลา  
อิสฺสนฺติ ทุเมฺมธา, คุณี นิโทฺทสการิโน;  
  
ครุโก ปณฺฑิโต เอตส-มิสฺสํ เตหฺยวิทฺวา สโมฯ  
</li>

</li>
  
<li>
เย วุฑฺฒมปจยนฺติ, นรา ธมฺมสฺส โกวิทา;

</li>
  
<li>

<li>

</li>
  
อาสีเสเถว  
ปุริโส, น นิพฺพิเนฺทยฺย ปณฺฑิโต;

</li>
  
<li>

<li>

</li>
  
<li>

</li>
  
<li>

</li>
  
<li>

</li>
  
<li>

วิสทญาณสฺส, ญาโต อโตฺถ ตรสฺสน;

</li>
  
<li>

</li>
  
<li>
ธีโร  
จ วิวิธานญฺญู, ปเรสํ วิวรานุคู;

</li>
  
<li>

เตโชยํ, มตฺติกํ น มุทุํ กเร;

</li>
  
<li>

<li>

</li>

<li>

</li>
  
<li>
วิทฺวตฺตญฺจ ราชตฺตญฺจ, เนว ตุลฺยํ กทาจิปิ;

</li>
  
<li>
สตํ

<li>

<li>

<li>

<li>

<li>

<li>

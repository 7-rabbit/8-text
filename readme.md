### Unicode Text Collection

This contains a Hebrew text collection in Unicode (UTF8), the Apocrypha text in unicode, the Talmud text in unicode, the Tanakh text in unicode, the Midrash text in unicode, and the Mishnah text in unicode.

It contains a Sanskrit collection in unicode too. The Rig Veda text in sanskrit unicode, the Mahabharata text in sanskrit unicode, and the Puranas text in sanskrit unicode.

It also contains a Tibetan text collection in unicode.

## Contributing

You can contribute in many ways:

1. Create a document translation donated to the public domain.
2. Find new original source, public domain documents in text format.
3. Cleanup document formatting.
4. Check correctness of documents against other sources.

### Creating a Translation

To create a translation, follow this protocol.

1. Fork this project.
2. Create a branch named after the document and the translation.
3. Push your first change.
4. Create a pull request when you first start, so we can keep track of whose working on what. Label it as WIP.
5. Remove the label when it's ready!

```bash
git checkout -b enneads/english
```
